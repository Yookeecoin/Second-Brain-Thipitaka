# แหล่ง-เจโตขีลสูตร (จาก thaimm vol-18)

ตัวบทพระสูตร (ไม่รวมอรรถกถา) มมร เล่ม 18 ข้อ ๒๒๖–๒๓๒ บรรทัด 4222–4469 ของ [[vol-18#หน้า 0171 (ข้อ 228)|vol-18 หน้า 0171 (ข้อ 228)]]

## จุดเด่นที่ใช้สรุป
- ตะปูตรึงใจ 5: สงสัยในศาสดา/ธรรม/สงฆ์/สิกขา + โกรธเพื่อนพรหมจรรย์
- เครื่องผูกพันใจ 5: ราคะในกาม/กาย/รูป + ติดสุขกิน-นอน + ประพฤติพรหมจรรย์เพื่อหวังเป็นเทพ
- ฝ่ายเจริญ: ละครบ → เพียร + อิทธิบาท + อุปมาแม่ไก่ฟักไข่

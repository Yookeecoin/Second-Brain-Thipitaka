# แหล่ง: จูฬตัณหาสังขยสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๓๗)

- ที่มา: `raw/tipitaka/thaimm/vol-19.md` ข้อ ๔๓๓–๔๓๙
- ฉบับ: มมร เล่ม 19 | ตำแหน่ง: มหายมกวรรค สูตรที่ 7

## ใจความสำคัญ
1. ที่ปราสาทมิคารมาตา บุพพาราม สาวัตถี ท้าวสักกะถามข้อปฏิบัติย่อสู่ความสิ้นตัณหา
2. คำตอบ: ได้สดับว่า "ธรรมทั้งปวงไม่ควรยึดมั่น" → รู้ยิ่ง/กำหนดรู้ → เห็นเวทนา (สุข/ทุกข์/อุเบกขา) ไม่เที่ยง หน่าย ดับ สละคืน → ไม่ยึด → ไม่สะดุ้ง → ปรินิพพานเฉพาะตน
3. พระโมคคัลลานะเหาะขึ้นดาวดึงส์ทดสอบ ท้าวสักกะพาชมเวชยันตปราสาท (ประมาท) → ทรงใช้นิ้วหัวแม่เท้ากดปราสาทให้สั่นสะเทือน ให้สังเวช
4. ท้าวสักกะสลดแล้วเล่าธรรมได้ถูก พระพุทธเจ้ายืนยันคำสอนอีกครั้ง

## จุดเด่น
- รวบยอดอนัตตา+วิปัสสนาในเวทนาในประโยคเดียว "ธรรมทั้งปวงไม่ควรยึดมั่น"
- ฉากเขย่าเวชยันตปราสาท = บทเรียนความไม่ประมาทแม้ในเทวดา

# สรุปแหล่ง: มหาปรินิพพานสูตร

แหล่งต้นฉบับ: [[vol-13#หน้า 0234 (ข้อ 67)|vol-13 หน้า 0234 (ข้อ 67)]] (หน้า ~?, ข้อ 67–144) | ที.ม. ๑๐
ที่อ้างอิง: พระสุตตันตปิฎก ทีฆนิกาย มหาวรรค, มหาปรินิพพานสูตร (สูตรที่ ๓) — **สูตรยาวที่สุดในพระไตรปิฎก** ว่าด้วยปัจฉิมกาลของ [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] จนเสด็จปรินิพพาน

## ลำดับเหตุการณ์สำคัญ
- เริ่มที่คิชฌกูฏ [[rajagaha (ราชคฤห์)|ราชคฤห์]]: [[ajatasattu (พระเจ้าอชาตศัตรู)|พระเจ้าอชาตศัตรู]] จะตีแคว้นวัชชี ส่งวัสสการพราหมณ์มาเฝ้า → ทรงแสดง **[[aparihaniya-dhamma-7 (อปริหานิยธรรม 7)|อปริหานิยธรรม 7]]** (ของเจ้าวัชชี แล้วของภิกษุ)
- เสด็จจาริกผ่าน [[nalanda (นาลันทา)|นาลันทา]] → ปาฏลิคาม (ทำนายความเจริญของปาฏลีบุตร) → ข้ามคงคา → [[vesali (เวสาลี)|เวสาลี]] (นางอัมพปาลีถวายสวน) → จำพรรษาสุดท้ายที่เวฬุวคาม ทรงประชวรหนัก ตรัส **"จงมีตนเป็นเกาะ มีธรรมเป็นที่พึ่ง" (อัตตทีปะ)**
- ทรงปลงพระชนมายุสังขารที่ปาวาลเจดีย์ (จะปรินิพพานในอีก 3 เดือน)
- เมือง [[pava (ปาวา)|ปาวา]]: ฉันสูกรมัททวะของ [[cunda (จุนทกัมมารบุตร)|นายจุนทกัมมารบุตร]] ทรงประชวรหนัก (ตรัสว่าบิณฑบาต 2 ครั้งมีผลมาก: ก่อนตรัสรู้ + ก่อนปรินิพพาน)
- [[kusinara (กุสินารา)|กุสินารา]] ระหว่างต้นสาละ: **[[subhadda (สุภัททปริพาชก)|สุภัททปริพาชก]] = ปัจฉิมสาวก** (สาวกองค์สุดท้ายที่ทรงบวชให้), ตรัส **"ธรรมและวินัยจะเป็นศาสดาแทนเรา"**, อนุญาตให้เพิกถอนสิกขาบทเล็กน้อย
- **[[pacchima-vaca (ปัจฉิมวาจา)|ปัจฉิมวาจา]]**: "สังขารทั้งหลายมีความเสื่อมไปเป็นธรรมดา จงยังความไม่ประมาทให้ถึงพร้อม" แล้วปรินิพพานผ่านฌานโดยลำดับ
- ทรงแสดง **[[sangvejaniya-4 (สังเวชนียสถาน 4)|สังเวชนียสถาน 4]]** (ประสูติ-ตรัสรู้-แสดงธรรมจักร-ปรินิพพาน)
- ถวายพระเพลิง แบ่งพระสรีรธาตุ: โทณพราหมณ์แบ่งเป็น 8 ส่วน

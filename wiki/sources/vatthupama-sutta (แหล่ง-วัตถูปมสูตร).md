# แหล่ง-วัตถูปมสูตร (จาก thaimm vol-17)

ตัวบทพระสูตร (ไม่รวมอรรถกถา) มมร เล่ม 17 ข้อ ๙๑–๙๙ บรรทัด 10400–10543 ของ [[vol-17#หน้า 0434 (ข้อ 94 95)|vol-17 หน้า 0434 (ข้อ 94 95)]]

## จุดเด่นที่ใช้สรุป
- อุปมาผ้า: จิตเศร้าหมอง→ทุคติ / ไม่เศร้าหมอง→สุคติ
- อุปกิเลส 16
- ลำดับ: รู้-ละอุปกิเลส → เลื่อมใสพระรัตนตรัย → ปราโมทย์-ปีติ-สุข-สมาธิ → พรหมวิหาร 4 → วิมุตติ
- สุนทริกพราหมณ์ + คาถาน้ำล้างบาปไม่ได้ → ขอบวช บรรลุอรหัต

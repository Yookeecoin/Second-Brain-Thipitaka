# สรุปแหล่ง: มหาลิสูตร

แหล่งต้นฉบับ: [[vol-12#หน้า 0094 (ข้อ 241 242)|vol-12 หน้า 0094 (ข้อ 241 242)]] (หน้า 93–102, ข้อ 239–255)
ที่อ้างอิง: พระสุตตันตปิฎก ทีฆนิกาย สีลขันธวรรค, มหาลิสูตร (สูตรที่ ๖) | ที.สี. ๙

## ใจความสำคัญ
- [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ประทับกูฏาคารสาลา ป่ามหาวัน เมือง [[vesali (เวสาลี)|เวสาลี]] เจ้าลิจฉวี [[mahali (โอฏฐัทธลิจฉวี)|โอฏฐัทธะ (มหาลิ)]] มาเฝ้า
- ถามกรณี สุนักขัตตะ เห็นทิพยรูปได้ 3 ปี แต่ฟังทิพยเสียงไม่ได้ ทรงอธิบายว่าเป็นเพราะ **อบรมสมาธิด้านเดียว** (อบรมเพื่อเห็นรูปอย่างเดียว → เห็นแต่ไม่ได้ยิน / อบรมทั้งสองส่วน → ได้ทั้งคู่)
- คำถามสำคัญ: ภิกษุประพฤติพรหมจรรย์เพื่อได้สมาธิ(ทิพยจักขุ-โสต)เหล่านี้หรือ? ตรัสว่า **ไม่ใช่** มีธรรมที่ยิ่งและประณีตกว่า คือ [[ariya-puggala-4 (อริยบุคคล 4)|อริยผล 4]]: โสดาบัน → สกทาคามี → อนาคามี → อรหันต์
- ทางดำเนินให้ถึง = [[majjhima-patipada (มัชฌิมาปฏิปทา)|มรรคมีองค์ 8]]
- ทรงเล่าเหตุการณ์ที่โกสัมพี (มัณฑิยปริพพาชก + ชาลิยะ ถาม "ชีวะกับสรีระเป็นอันเดียวกันหรือต่างกัน") ทรงชี้ว่าผู้รู้แจ้งตามลำดับฝึก (อ้าง [[samannaphala-sutta-page (สามัญญผลสูตร)|สามัญญผลสูตร]]) ย่อมไม่มัวพยากรณ์ปัญหานี้ (พักไว้ ไม่เป็นไปเพื่อความหลุดพ้น)

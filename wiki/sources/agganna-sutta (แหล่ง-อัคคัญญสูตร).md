# สรุปแหล่ง: อัคคัญญสูตร

แหล่งต้นฉบับ: [[raw/tipitaka/thaimm/vol-15#หน้า 0145 (ข้อ 51)|vol-15 หน้า 145 ข้อ 51–64]] | ที.ปา. ๑๑
ที่อ้างอิง: พระสุตตันตปิฎก ทีฆนิกาย ปาฏิกวรรค, อัคคัญญสูตร (สูตรที่ ๔) — "สูตรว่าด้วยปฐมกาล"

## ใจความสำคัญ
- [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ประทับบุพพาราม (มิคารมาตุปราสาท) [[savatthi (สาวัตถี)|สาวัตถี]] [[vasettha-bharadvaja (วาเสฏฐะและภารัทวาชะ)|วาเสฏฐะและภารัทวาชะ]] (บวชเป็นสามเณรแล้ว) ถูกพราหมณ์ด่าว่าละทิ้งวรรณะสูงสุด อ้างว่า "พราหมณ์เกิดจากพรหม เป็นทายาทพรหม"
- ทรงหักความถือตัวเรื่อง **[[vanna-4 (วรรณะ 4)|วรรณะ 4]]**: ทั้ง 4 วรรณะทำดี-ชั่วได้เหมือนกัน
- ทรงเล่า **กำเนิดโลก-สังคม**: หลังโลกพินาศ-ก่อตัวใหม่ สัตว์เป็น [[loka-uppatti|อาภัสสรพรหม]] มีรัศมี กินง่ายดูง่าย แล้วค่อยๆ หยาบลงเพราะตัณหา (ง้วนดิน/รสปฐวี → ข้าวสาลี → สะสม → ลักขโมย) จนต้องตกลงเลือกผู้รักษาความเป็นธรรม
- **มหาสมมต**: มหาชน "สมมติ (เลือก)" ผู้มีคุณสมบัติให้ว่ากล่าว-ติเตียน-ขับไล่คนผิด แลกกับส่วนข้าวสาลี → เกิดคำ **"มหาสมมติ" → "กษัตริย์" (เป็นใหญ่แห่งเขต) → "ราชา" (ผู้ทำให้ผู้อื่นยินดีโดยธรรม)** = กำเนิดกษัตริย์โดย **สัญญาประชาคม-หน้าที่ ไม่ใช่พระเจ้าสร้าง**; พราหมณ์ก็เกิดจากผู้ "ลอยบาป" (ฌายี) — ทุกวรรณะเกิดตามหน้าที่
- บทสรุป: **"ธรรมเท่านั้นประเสริฐที่สุดในหมู่ชน ทั้งปัจจุบันและภพหน้า"** — ผู้บรรลุธรรม (อรหันต์) จากวรรณะใดก็ประเสริฐสุด

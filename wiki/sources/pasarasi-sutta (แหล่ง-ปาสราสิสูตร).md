# แหล่ง-ปาสราสิสูตร / อริยปริเยสนสูตร (จาก thaimm vol-18)

ตัวบทพระสูตร (ไม่รวมอรรถกถา) มมร เล่ม 18 ข้อ ๓๑๒–๓๒๘ บรรทัด 10231–10760 ของ [[vol-18#หน้า 0410 (ข้อ 312)|vol-18 หน้า 0410 (ข้อ 312)]]

## จุดเด่นที่ใช้สรุป
- การแสวงหา 2 อย่าง: อนริยปริเยสนา / อริยปริเยสนา
- ก่อนตรัสรู้: ไปหาอาฬารดาบส (อากิญจัญฯ) + อุททกดาบส (เนวสัญญาฯ) ทั้ง 2 ไม่นำสู่นิพพาน → ลา
- ถึงอุรุเวลา → บำเพ็ญเพียร → ตรัสรู้ครบ 6 ขั้น
- ตรึกว่าจะแสดงธรรมไหม → สหัมบดีพรหมอาราธนา 3 ครั้ง
- ตรวจดูโลก = อุปมาบัว 3 ระดับ
- ระลึกถึงปัญจวัคคีย์ที่อิสิปตนะ พบอุปกอาชีวก
- โปรดปัญจวัคคีย์ → ปฐมเทศนา → บรรลุอรหันต์
- ปิดด้วยฌาน 8 + นิโรธสมาบัติ = ที่มารไปไม่ถึง

# แหล่ง-จูฬหัตถิปโทปมสูตร (จาก thaimm vol-18)

ตัวบทพระสูตร (ไม่รวมอรรถกถา) มมร เล่ม 18 ข้อ ๓๒๙–๓๓๙ บรรทัด 11774–12144 ของ [[vol-18#หน้า 0470 (ข้อ 329)|vol-18 หน้า 0470 (ข้อ 329)]]

## จุดเด่นที่ใช้สรุป
- ปิโลติกปริพาชกชมพระพุทธเจ้าด้วย "ร่องรอย 4" (ขัตติย/พราหมณ/คฤหบดี/สมณบัณฑิตเข้าหาก็กลายเป็นสาวก)
- ทรงสอนว่ายังไม่บริบูรณ์ ต้องเห็นช้างเต็มตัว
- แผนที่ปฏิบัติ: ออกบวช + ศีล + อินทรียสังวร + สติสัมป + สันโดษ
- ละนิวรณ์ → รูปฌาน 4 → ปุพเพนิวาส → จุตูปปาต → อาสวักขย
- แต่ละขั้นทรงเรียก "ตถาคตบท" (รอยเท้าตถาคต) — ยังไม่ตกลงใจ
- ตกลงใจจริงตอนอาสวักขย (เห็นช้างเต็มตัว)
- ชาณุสโสณิเป็นอุบาสกตลอดชีวิต

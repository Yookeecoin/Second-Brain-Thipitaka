# แหล่ง: โปตลิยสูตร (มัชฌิมนิกาย มัชฌิมปัณณาสก์ สูตรที่ ๕๔)

- ที่มา: `raw/tipitaka/thaimm/vol-20.md` ข้อ ๓๖–๕๕
- ฉบับ: มมร เล่ม 20 | ตำแหน่ง: คหปติวรรค สูตรที่ 4

## ใจความสำคัญ
1. โปตลิยคฤหบดี (สละการงาน มอบทรัพย์ให้บุตร) ไม่พอใจถูกเรียก "คฤหบดี" อ้างว่าตัดโวหารแล้ว
2. พระพุทธเจ้าชี้ว่าการตัดโวหารในอริยวินัยต่างกัน → ธรรมตัดโวหาร 8: ละปาณาติบาต/อทินนาทาน/มุสา/ปิสุณา/โลภ/โกรธ(นินทา)/คับแค้น/ดูหมิ่น
3. อุปมาโทษกาม 7: ร่างกระดูก / ชิ้นเนื้อ / คบเพลิงทวนลม / หลุมถ่านเพลิง / ความฝัน / ของยืม / ผลไม้บนต้น
4. เห็นโทษกาม → เจริญอุเบกขาเอกัตตะ → ดับความยึดโลกามิส → (วิชชา 3 สิ้นอาสวะ)

## จุดเด่น
- "ตัดโวหาร" แท้ = ตัดกิเลส/สังโยชน์ ไม่ใช่แค่เลิกการงาน
- ชุดอุปมากาม 7 อย่าง — ภาพคลาสสิกแสดงโทษของกาม

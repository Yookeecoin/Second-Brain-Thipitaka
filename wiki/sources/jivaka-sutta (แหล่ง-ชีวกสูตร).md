# แหล่ง: ชีวกสูตร (มัชฌิมนิกาย มัชฌิมปัณณาสก์ สูตรที่ ๕๕)

- ที่มา: `raw/tipitaka/thaimm/vol-20.md` ข้อ ๕๖–๖๑
- ฉบับ: มมร เล่ม 20 | ตำแหน่ง: คหปติวรรค สูตรที่ 5

## ใจความสำคัญ
1. หมอชีวกถามที่อัมพวันของตน ราชคฤห์: จริงหรือที่ทรงเสวยเนื้อที่เขาฆ่าเจาะจงถวาย
2. เนื้อไม่ควรฉัน = เห็น/ได้ยิน/รังเกียจว่าฆ่าเจาะจงเพื่อตน; เนื้อควรฉัน = ไม่เห็น/ไม่ได้ยิน/ไม่รังเกียจ (ติโกฏิปริสุทธิ)
3. ภิกษุผู้แผ่เมตตา-กรุณา-มุทิตา-อุเบกขา ฉันบิณฑบาตโดยไม่ติด ไม่สยบ → "อาหารไม่มีโทษ"
4. ผู้สั่งฆ่าเจาะจงถวาย → บาป 5 สถาน (สั่งนำมา / สัตว์ถูกลากมาทุกข์ / สั่งฆ่า / สัตว์ถูกฆ่าทุกข์ / ทำให้ฉันของอกัปปิยะ)
5. ชีวกถึงสรณะเป็นอุบาสก

## จุดเด่น
- หลักพุทธานุญาตการฉันเนื้อ (เนื้อบริสุทธิ์ 3 ส่วน) — ต่างจากข้อเสนอเทวทัตที่ขอห้ามเด็ดขาด
- เจตนาฆ่าเป็นบาป ไม่ใช่การกิน

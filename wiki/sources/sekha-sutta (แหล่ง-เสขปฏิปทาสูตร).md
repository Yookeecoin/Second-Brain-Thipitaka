# แหล่ง: เสขปฏิปทาสูตร (มัชฌิมนิกาย มัชฌิมปัณณาสก์ สูตรที่ ๕๓)

- ที่มา: `raw/tipitaka/thaimm/vol-20.md` ข้อ ๒๔–๓๕
- ฉบับ: มมร เล่ม 20 | ตำแหน่ง: คหปติวรรค สูตรที่ 3

## ใจความสำคัญ
1. เจ้าศากยะเชิญพระพุทธเจ้าขึ้นสัณฐาคารใหม่ที่นิโครธาราม กบิลพัสดุ์; ทรงให้พระอานนท์แสดงเสขปฏิปทาแก่เจ้ามหานามะแทน
2. องค์: ศีล + อินทรียสังวร + โภชเนมัตตัญญุตา + ชาคริยานุโยค + สัปปุริสธรรม 7 (ศรัทธา/หิริ/โอตตัปปะ/พหูสูต/วิริยะ/สติ/ปัญญา) + ฌาน 4
3. อุปมาลูกไก่เจาะเปลือกไข่ → จากจตุตถฌาน → วิชชา 3
4. วิชชา-จรณะ; ยกคาถาสนังกุมารพรหม "ผู้ถึงพร้อมวิชชาจรณะประเสริฐสุด"; พระพุทธเจ้ารับรอง "สาธุ"

## จุดเด่น
- แม่บท "เสขปฏิปทา" (ทางผู้ยังศึกษา) = จรณะ 15 โดยนัย
- บทบาทพระอานนท์แสดงธรรมแทนพระพุทธเจ้า

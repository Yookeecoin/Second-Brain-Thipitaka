# แหล่ง: มหาเวทัลลสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๔๓)

- ที่มา: `raw/tipitaka/thaimm/vol-19.md` ข้อ ๔๙๓–๕๐๔
- ฉบับ: มมร เล่ม 19 | ตำแหน่ง: จูฬยมกวรรค สูตรที่ 3
- หมายเหตุ: ในต้นฉบับมมรท้ายสูตรพิมพ์ว่า "ที่ ๗" (น่าจะคลาดเคลื่อน) — ตามลำดับวรรคคือสูตรที่ 3 ของจูฬยมกวรรค = สูตรที่ 43 ของมัชฌิมนิกาย

## ใจความสำคัญ (ถาม-ตอบ ระหว่างมหาโกฏฐิกะ–สารีบุตร)
1. ปัญญาทราม/ปัญญา = ไม่รู้/รู้ชัดอริยสัจ; วิญญาณ = รู้แจ้งเวทนา
2. ปัญญา-วิญญาณ-เวทนา-สัญญา "คละกัน แยกไม่ออก" — ต่างกันที่กิจ: ปัญญาควรเจริญ, วิญญาณควรกำหนดรู้
3. ปัจจัยสัมมาทิฏฐิ 2: ปรโตโฆสะ + โยนิโสมนสิการ; องค์ 5 หนุนสัมมาทิฏฐิ: ศีล/สุตะ/สากัจฉา/สมถะ/วิปัสสนา
4. ภพ 3; ภพใหม่เกิดเพราะอวิชชา+ตัณหา ดับเพราะสำรอกอวิชชา-ดับตัณหา
5. อายุ-ไออุ่น-วิญญาณ อิงกัน; คนตาย vs ผู้เข้านิโรธสมาบัติ (ตาย=อายุหมด ไออุ่นดับ; นิโรธ=อายุยังอยู่ ไออุ่นยังอยู่ อินทรีย์ผ่องใส)
6. เจโตวิมุตติ 4: อัปปมาณา (พรหมวิหาร) / อากิญจัญญา / สุญญตา / อนิมิตตา — โดยใจความ อกุปปาเจโตวิมุตติ (ไม่กำเริบ) เป็นเลิศ เพราะว่างจากราคะ-โทสะ-โมหะ

## จุดเด่น
- สูตร "อภิธรรมเชิงสนทนา" นิยามนามธรรมหลักและความสัมพันธ์
- คำอธิบายคลาสสิก "คนตาย vs ผู้เข้านิโรธสมาบัติ"
- คู่กับจูฬเวทัลลสูตร (สูตรที่ 44)

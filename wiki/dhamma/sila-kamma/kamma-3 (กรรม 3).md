# กรรม 3 (Tīṇi kammāni) — กรรมทางทวาร 3

การกระทำที่ประกอบด้วย**เจตนา** จำแนกตามทวาร 3: **กายกรรม / วจีกรรม / มโนกรรม**

- พระพุทธเจ้าทรงวาง **มโนกรรม (ใจ) มีโทษมากที่สุด** เพราะ **เจตนาคือตัวกรรม** ("เจตนาหํ ภิกฺขเว กมฺมํ วทามิ") — ใจเป็นต้นทางของกาย-วาจา
- ต่างจาก[[nigantha-nataputta (นิครนถ์นาฏบุตร)|นิครนถ์]]ที่จัดเป็น **ทัณฑะ 3** และว่ากายทัณฑะมากสุด (เน้นการกระทำภายนอก) — ดู[[upali-sutta-page (อุปาลิวาทสูตร)|อุปาลิวาทสูตร]]
- เชื่อมกับ[[kammapatha-10 (กรรมบถ 10)|กรรมบถ 10]] (แจกกาย 3/วาจา 4/ใจ 3) และ[[kamma-4 (กรรมดำกรรมขาว 4)|กรรม 4]] (จำแนกตามวิบาก)

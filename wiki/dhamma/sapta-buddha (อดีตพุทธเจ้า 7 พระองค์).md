# อดีตพุทธเจ้า 7 พระองค์

จาก [[mahapadana-sutta-page (มหาปทานสูตร)|มหาปทานสูตร]]: [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ทรงเล่าถึงพระสัมมาสัมพุทธเจ้า 7 พระองค์ (รวมพระองค์เอง) ตามลำดับกัป

| พระพุทธเจ้า | กัป | ชาติ/โคตร |
|---|---|---|
| **วิปัสสี** ([[vipassi (พระวิปัสสีพุทธเจ้า)|หน้า]]) | 91 กัปก่อน | กษัตริย์ / โกณฑัญญโคตร |
| **สิขี** | 31 กัปก่อน | กษัตริย์ / โกณฑัญญโคตร |
| **เวสสภู** | 31 กัปก่อน | กษัตริย์ / โกณฑัญญโคตร |
| **กกุสันธะ** | ภัททกัปนี้ | พราหมณ์ / กัสสปโคตร |
| **โกนาคมนะ** | ภัททกัปนี้ | พราหมณ์ / กัสสปโคตร |
| **กัสสปะ** | ภัททกัปนี้ | พราหมณ์ / กัสสปโคตร |
| **โคตมะ** (พระพุทธเจ้าองค์ปัจจุบัน) | ภัททกัปนี้ | กษัตริย์ / โคตมโคตร |

> สูตรยังระบุรายละเอียดอื่นของแต่ละพระองค์ (อายุขัย ไม้โพธิ์ อัครสาวก พุทธอุปัฏฐาก พระชนก-ชนนี เมืองหลวง) — แสดงว่าความเป็นพุทธะมี "ธรรมดา/แบบแผน" ร่วมกัน

# ศีล 3 ระดับ — จุลศีล / มัชฌิมศีล / มหาศีล

หมวดศีลที่ [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ทรงแสดงใน [[brahmajala-sutta-page (พรหมชาลสูตร)|พรหมชาลสูตร]] ในฐานะสิ่งที่ปุถุชนมักใช้สรรเสริญตถาคต แต่ทรงชี้ว่า "มีประมาณน้อย ยังต่ำ เป็นเพียงศีล" เทียบกับปัญญารู้แจ้ง [[ditthi-62 (ทิฏฐิ 62)|ทิฏฐิ 62]]

- **จุลศีล** (ศีลเล็กน้อย) — เว้นจากการฆ่าสัตว์ ลักทรัพย์ เมถุน พูดเท็จ ส่อเสียด ฯลฯ (ระดับพื้นฐานของการละเว้นทุจริต)
- **มัชฌิมศีล** (ศีลกลาง) — เว้นจากการประกอบสิ่งที่ไม่ควรในวงกว้างขึ้น เช่น พรากพืชคาม การสะสมบริโภค การดูการเล่นต่างๆ
- **มหาศีล** (ศีลใหญ่) — เว้นจากมิจฉาชีพ/ดิรัจฉานวิชา เช่น ทำนายทายทัก ดูฤกษ์ หมอเวทมนตร์

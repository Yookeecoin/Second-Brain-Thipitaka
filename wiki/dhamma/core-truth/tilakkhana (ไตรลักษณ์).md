# ไตรลักษณ์ (tilakkhaṇa)

ลักษณะสามัญ 3 ประการของสังขตธรรม/[[khandha-5 (ขันธ์ 5)|ขันธ์ 5]] ที่อริยสาวกพิจารณาเห็นเพื่อเบื่อหน่ายคลายกำหนัด

- **[[anicca (อนิจจัง)|อนิจจัง]]** — ไม่เที่ยง เกิด-ดับ แปรปรวน
- **ทุกขัง** — เป็นทุกข์ ทนอยู่ไม่ได้ ถูกบีบคั้น
- **[[anatta (อนัตตา)|อนัตตา]]** — ไม่ใช่ตัวตน บังคับบัญชาไม่ได้
- แม่บทพิจารณา: "สิ่งใดไม่เที่ยง สิ่งนั้นเป็นทุกข์ สิ่งใดเป็นทุกข์ สิ่งนั้นเป็นอนัตตา → นั่นไม่ใช่ของเรา ไม่เป็นเรา ไม่ใช่ตัวตนของเรา"
- เห็นไตรลักษณ์ → [[nibbida (นิพพิทา)|นิพพิทา]] → [[viraga (วิราคะ)|วิราคะ]] → [[vimutti (วิมุตติ)|วิมุตติ]]; เป็นแกนของขันธสังยุตทั้งหมด

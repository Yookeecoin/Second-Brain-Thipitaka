# ปฏิจจสมุปบาท (อิทัปปัจจยตา — สิ่งทั้งหลายเกิดเพราะอาศัยเหตุปัจจัย)

ธรรมที่ [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] (และอดีตพุทธเจ้าเช่น [[vipassi (พระวิปัสสีพุทธเจ้า)|พระวิปัสสี]]) ตรัสรู้ ปรากฏใน [[mahapadana-sutta-page (มหาปทานสูตร)|มหาปทานสูตร]]

## สายเกิด (อนุโลม — 12 องค์เต็ม)
[[avijja (อวิชชา)|อวิชชา]] → [[sankhara (สังขาร)|สังขาร]] → [[vinnana (วิญญาณ)|วิญญาณ]] → [[namarupa (นามรูป)|นามรูป]] → [[salayatana (อายตนะ 6)|สฬายตนะ]] → [[phassa (ผัสสะ)|ผัสสะ]] → [[vedana (เวทนา)|เวทนา]] → [[tanha (ตัณหา)|ตัณหา]] → [[upadana-4 (อุปาทาน 4)|อุปาทาน]] → [[bhava-3 (ภพ 3)|ภพ]] → [[jati (ชาติ)|ชาติ]] → [[jaramarana (ชรามรณะ)|ชรามรณะ]] + โสกปริเทวทุกขโทมนัสอุปายาส = กองทุกข์ทั้งมวล

> หมายเหตุ: ในมหาปทานสูตร/มหานิทานสูตร การสาวย้อน **สิ้นสุดที่ "วิญญาณ ↔ นามรูป"** ไม่ได้สาวถึงสังขาร-อวิชชา — สายเต็ม 12 องค์ แสดงใน [[nidana-buddhavagga-page (นิทานสังยุต-พุทธวรรค)|นิทานสังยุต พุทธวรรค]] (SN 12.1–12.2) และที่อื่นในนิทานวรรค

## สายดับ (ปฏิโลม)
เพราะอวิชชาดับด้วยการสำรอกโดยไม่เหลือ สังขารจึงดับ → วิญญาณ → นามรูป → สฬายตนะ → ผัสสะ → เวทนา → ตัณหา → อุปาทาน → ภพ → ชาติ → ชรามรณะ โสก ฯลฯ ดับ = ความดับทุกข์ทั้งมวล

## นิยามมาตรฐานแต่ละองค์ (จากวิภังคสูตร SN 12.2)
- **อวิชชา** = ความไม่รู้ใน[[ariyasacca-4 (อริยสัจ 4)|อริยสัจ 4]] ทั้งสี่ (ทุกข์/สมุทัย/นิโรธ/มรรค)
- **สังขาร** = กายสังขาร วจีสังขาร จิตตสังขาร
- **วิญญาณ** = วิญญาณ 6 (จักขุ โสต ฆาน ชิวหา กาย มโน)
- **นามรูป** = นาม (เวทนา สัญญา เจตนา ผัสสะ มนสิการ) + รูป (มหาภูต 4 + รูปอาศัยมหาภูต)
- **สฬายตนะ** = อายตนะ 6 (ตา หู จมูก ลิ้น กาย ใจ)
- **ผัสสะ** = ผัสสะ 6 (จักขุสัมผัส โสต ฆาน ชิวหา กาย มโนสัมผัส)
- **เวทนา** = เวทนา 6 (เวทนาที่เกิดจากสัมผัสทาง 6 ทวาร)
- **ตัณหา** = ตัณหา 6 (รูป สัทท คันธ รส โผฏฐัพพ ธัมมตัณหา)
- **อุปาทาน** = อุปาทาน 4 (กามุปาทาน ทิฏฐุปาทาน สีลพัตตุปาทาน อัตตวาทุปาทาน)
- **ภพ** = ภพ 3 (กามภพ รูปภพ อรูปภพ)
- **ชาติ** = ความเกิด ความก่อเกิด ความหยั่งลง ปรากฏแห่งขันธ์ ได้อายตนะครบ
- **ชราและมรณะ** = ชรา (ฟันหลุด ผมหงอก หนังเกลียว อินทรีย์เสื่อม) + มรณะ (ความแตกแห่งขันธ์ ขาดชีวิตินทรีย์)

## เชื่อมโยง
- เป็นเหตุผลเบื้องหลังที่ทรงไม่ตอบ [[avyakata-10 (อัพยากตปัญหา 10)|อัพยากตปัญหา]]/[[jiva-sarira (ชีวะกับสรีระ)|ชีวะ-สรีระ]] (ทุกอย่างเป็นเหตุปัจจัย ไม่มีอัตตาเที่ยง)
- เป็นแกนเดียวกับ [[ariyasacca-4 (อริยสัจ 4)|อริยสัจ 4]] (สมุทัย-นิโรธ)
- ปรากฏใน [[brahmajala-sutta-page (พรหมชาลสูตร)|พรหมชาลสูตร]] ด้วย (ทิฏฐิเกิดเพราะผัสสะเป็นปัจจัย)

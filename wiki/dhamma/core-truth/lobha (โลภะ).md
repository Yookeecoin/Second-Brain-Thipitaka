# โลภะ

**ความโลภ/อยากได้** — 1 ใน[[akusala-mula-3 (อกุศลมูล 3)|อกุศลมูล 3]] (โลภะ-โทสะ-โมหะ) รากเหง้าของอกุศลทั้งปวง

- ใกล้เคียง[[raga (ราคะ)|ราคะ]] (ความกำหนัด/ติดใจ) — โลภะเป็นมูลให้เกิดทุจริต (ลัก/โกง/เพ่งเล็งของผู้อื่น = อภิชฌา)
- ตรงข้ามกับ **อโลภะ** (ความไม่โลภ = เนกขัมมะ/จาคะ/ทาน)
- ละด้วยการเห็นโทษ + [[dana (ทาน)|ทาน]]/[[asubha (อสุภะ)|อสุภะ]] + ปัญญา

โยง[[raga (ราคะ)|ราคะ]] · [[tanha (ตัณหา)|ตัณหา]] · [[akusala-mula-3 (อกุศลมูล 3)|อกุศลมูล 3]]

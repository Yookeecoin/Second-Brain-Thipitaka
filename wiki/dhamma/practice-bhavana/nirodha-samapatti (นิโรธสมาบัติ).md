# นิโรธสมาบัติ / สัญญาเวทยิตนิโรธ (Saññāvedayitanirodha)

สมาบัติขั้นสูงสุด = **การดับสัญญาและเวทนา** (ดับการกำหนดหมายและการเสวยอารมณ์) ชั่วคราว เข้าถึงได้เฉพาะ[[ariya-puggala-4 (อริยบุคคล 4)|พระอนาคามีและพระอรหันต์]]ผู้ได้[[arupa-jhana-4 (อรูปฌาน 4)|สมาบัติ 8]] ครบ

## ลำดับการเข้าถึง
ผ่าน[[jhana-4 (ฌาน 4)|ฌาน 4]] → [[arupa-jhana-4 (อรูปฌาน 4)|อรูปฌาน 4]] (อากาสานัญจ./วิญญาณัญจ./อากิญจัญญ./เนวสัญญานาสัญญ.) → **ก้าวพ้นเนวสัญญานาสัญญายตนะ → สัญญาเวทยิตนิโรธ**

## ลักษณะสำคัญ (จาก[[mahavedalla-sutta-page (มหาเวทัลลสูตร)|มหาเวทัลลสูตร]] + [[culavedalla-sutta-page (จูฬเวทัลลสูตร)|จูฬเวทัลลสูตร]])
- **ลำดับดับสังขาร**: เข้า — วจีสังขาร (วิตกวิจาร) ดับก่อน → กายสังขาร (ลมหายใจ) → จิตตสังขาร (สัญญา-เวทนา); ออก — จิตตสังขารเกิดก่อน → กายสังขาร → วจีสังขาร
- **ต่างจากคนตาย**: คนตาย — อายุหมด ไออุ่น (เตโชธาตุ) ดับ อินทรีย์พัง; ผู้เข้านิโรธ — **อายุยังอยู่ ไออุ่นยังอยู่ อินทรีย์ผ่องใส**
- ออกจากนิโรธ — ผัสสะ 3 กระทบ: สุญญตผัสสะ / อนิมิตตผัสสะ / อัปปณิหิตผัสสะ; จิตน้อมไปสู่**วิเวก**

## ที่ปรากฏ
เป็นยอดของ "วิหารสมาบัติ" และ "ธรรมเครื่องอยู่สำราญ" ในหลายสูตร เช่น[[culagosinga-sutta-page (จูฬโคสิงคสาลสูตร)|จูฬโคสิงคสาลสูตร]], [[culasaropama-sutta-page (จูฬสาโรปมสูตร)|จูฬสาโรปมสูตร]] — เมื่อออกแล้ว "เห็นด้วยปัญญา อาสวะสิ้นไป"

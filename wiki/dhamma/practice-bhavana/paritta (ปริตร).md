# ปริตร (พระปริตร)

**พระสูตร/คาถาที่นิยมสวดเพื่อความคุ้มครองสวัสดี** ("ปริตฺต" = เครื่องป้องกัน, ของคุ้มกัน) — เป็นพุทธพจน์ที่[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]หรือพระสาวกแสดงไว้ในโอกาสต่างๆ แล้วภายหลังถูกรวบรวมมาสวดสาธยาย

- **หลักการ:** อานุภาพมาจากการระลึกถึงพระรัตนตรัย + สัจวาจา + เมตตา ไม่ใช่มนตร์ขลังภายนอก — สอดคล้องกับธรรม ไม่ใช่ไสยศาสตร์
- **พระสูตรที่ใช้เป็นปริตรในคลังนี้:**
  - **รตนสูตร** — แสดงคุณพระรัตนตรัย ใช้ระงับภัย 3 ที่[[vesali (เวสาลี)|เวสาลี]] (ดู[[pansa-02 (พรรษา 2)|พรรษา 2]])
  - [[metta (เมตตา)|กรณียเมตตสูตร]] — เจริญเมตตา บรรเทาความหวาดกลัว (ดู[[pansa-13 (พรรษา 13)|พรรษา 13]])
  - [[mangala-38 (มงคล 38)|มงคลสูตร]] — มงคลอันสูงสุด 38 ประการ
  - [[mahasamaya-sutta-page (มหาสมัยสูตร)|มหาสมัยสูตร]] — เทวดาหมื่นโลกธาตุมาประชุม
  - [[atanatiya-sutta-page (อาฏานาฏิยสูตร)|อาฏานาฏิยสูตร]] — ท้าวจาตุมหาราชถวายปริตรคุ้มครองพุทธบริษัท
  - **โพชฌงคปริตร** — จาก[[bojjhanga-7 (โพชฌงค์ 7)|โพชฌงค์ 7]] (เคยสวดโปรดผู้ป่วยให้หาย)
- ในไทยนิยมรวมเป็นชุด "เจ็ดตำนาน / สิบสองตำนาน" สวดในงานมงคล

# วิชชา 3 (Tevijjā — ความรู้แจ้ง 3)

ญาณ 3 ที่[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ทรงบรรลุในคืนตรัสรู้ หลังได้ [[jhana-4 (ฌาน 4)|ฌาน 4]] เกิดตามลำดับ 3 ยามของราตรี:

1. **ปุพเพนิวาสานุสสติญาณ** — ระลึกชาติก่อนได้ (ปฐมยาม)
2. **จุตูปปาตญาณ** (ทิพยจักษุ) — เห็นสัตว์จุติ-อุบัติเป็นไปตามกรรม สุจริต→สุคติ ทุจริต→ทุคติ (มัชฌิมยาม)
3. **อาสวักขยญาณ** — รู้ [[ariyasacca-4 (อริยสัจ 4)|อริยสัจ 4]] ตามจริง จิตหลุดพ้นจากอาสวะ (กาม/ภว/อวิชชา) "ชาติสิ้นแล้ว พรหมจรรย์อยู่จบแล้ว" (ปัจฉิมยาม)

แต่ละวิชชา "กำจัดอวิชชา วิชชาก็เกิด กำจัดความมืด ความสว่างก็เกิด"

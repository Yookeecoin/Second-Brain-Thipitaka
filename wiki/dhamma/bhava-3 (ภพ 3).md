# ภพ 3 (Tayo bhavā) — ภพ 3 ระดับ

ภพ = ภาวะความมี/ความเป็น ที่สัตว์เข้าถึงตามกรรม มี 3 ระดับ (จาก[[mahavedalla-sutta-page (มหาเวทัลลสูตร)|มหาเวทัลลสูตร]])

1. **กามภพ** — ภพของผู้ยังเสพ[[kamaguna-5 (กามคุณ 5)|กาม]] (อบาย-มนุษย์-เทวดา 6 ชั้น)
2. **รูปภพ** — ภพของรูปพรหม (ผู้ได้[[jhana-4 (ฌาน 4)|รูปฌาน]])
3. **อรูปภพ** — ภพของอรูปพรหม (ผู้ได้[[arupa-jhana-4 (อรูปฌาน 4)|อรูปฌาน]])

## ความสัมพันธ์
- ใน[[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]]: อุปาทาน → **ภพ** → ชาติ → ชรามรณะ
- การเกิดภพใหม่: เกิดเพราะ**อวิชชากั้น + ตัณหาผูก + เพลินอารมณ์**; ดับภพใหม่ = สำรอกอวิชชา ดับตัณหา (มหาเวทัลลสูตร)

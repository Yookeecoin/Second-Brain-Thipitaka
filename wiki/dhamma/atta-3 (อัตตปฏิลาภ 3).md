# อัตตปฏิลาภ 3 — การได้อัตตภาพ 3 แบบ

จาก [[potthapada-sutta-page (โปฏฐปาทสูตร)|โปฏฐปาทสูตร]] [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ทรงจำแนก "การได้อัตตภาพ" (อัตตา/ตัวตน ที่สัตว์ยึด) เป็น 3 ระดับ:

1. **โอฬาริก-อัตตภาพ** — อัตตภาพหยาบ (มีรูปกาย กินอาหาร)
2. **มโนมย-อัตตภาพ** — อัตตภาพสำเร็จด้วยใจ (มีรูปแต่ละเอียด)
3. **อรูป-อัตตภาพ** — อัตตภาพไม่มีรูป

## แก่น
- ทั้ง 3 มีได้ **ทีละอย่าง** (ขณะหนึ่งเป็นจริงแบบเดียว อีกสองเป็นโมฆะ) อุปมา นมสด→นมส้ม→เนยข้น→เนยใส
- ที่สุดแล้วชื่อเหล่านี้เป็นเพียง **โลกสมัญญา โลกนิรุตติ โลกโวหาร โลกบัญญัติ** ที่ตถาคต "ยืมมาพูด" โดยไม่ยึด (ทางปรมัตถ์ไม่มีตัวตนให้เกาะ → นัยอนัตตา)
- โยงกับ [[sanna (สัญญา)|สัญญา]] ที่เกิด-ดับตามเหตุ และการไม่พยากรณ์ [[avyakata-10 (อัพยากตปัญหา 10)|อัพยากตปัญหา]]

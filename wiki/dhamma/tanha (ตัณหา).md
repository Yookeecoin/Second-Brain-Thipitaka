# ตัณหา (Taṇhā) — ความทะยานอยาก

ความอยากที่เป็น**เหตุให้เกิดทุกข์** (ทุกขสมุทัยใน[[ariyasacca-4 (อริยสัจ 4)|อริยสัจ 4]]) และเป็นข้อต่อสำคัญใน[[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]] (เวทนา → **ตัณหา** → อุปาทาน → ภพ)

## ตัณหา 3
- **กามตัณหา** — อยากในกาม (รูป เสียง กลิ่น รส โผฏฐัพพะ)
- **ภวตัณหา** — อยากเป็น/อยากมี/อยากเกิดในภพ
- **วิภวตัณหา** — อยากไม่เป็น/อยากดับสูญ

## ความสำคัญ
- ใน[[culavedalla-sutta-page (จูฬเวทัลลสูตร)|จูฬเวทัลลสูตร]] ระบุ "ตัณหา 3 = สักกายสมุทัย" (เหตุเกิดแห่งสักกายะ); ความดับตัณหาโดยไม่เหลือ = สักกายนิโรธ
- ความสิ้นตัณหา (ตัณหักขยะ) = เป้าหมายสูงสุด ([[culatanhasankhaya-sutta-page (จูฬตัณหาสังขยสูตร)|จูฬตัณหาสังขยสูตร]], [[mahatanhasankhaya-sutta-page (มหาตัณหาสังขยสูตร)|มหาตัณหาสังขยสูตร]]): "ธรรมทั้งปวงไม่ควรยึดมั่น" → ไม่เพลินเวทนา → ตัณหาดับ
- เป็นรากของ[[ahara-4 (อาหาร 4)|อาหาร 4]] ที่ค้ำจุนสัตว์

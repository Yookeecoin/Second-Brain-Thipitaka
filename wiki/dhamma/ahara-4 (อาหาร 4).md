# อาหาร 4 (Cattāro āhārā) — เครื่องค้ำจุนการดำรงอยู่ของสัตว์

ปัจจัยที่ค้ำจุนสัตว์ผู้เกิดแล้วให้ดำรงอยู่ และอนุเคราะห์ผู้แสวงหาภพให้เกิด มี 4 อย่าง (จาก[[mahatanhasankhaya-sutta-page (มหาตัณหาสังขยสูตร)|มหาตัณหาสังขยสูตร]])

1. **กวฬิงการาหาร** — อาหารคือคำข้าว (หยาบ/ละเอียด) ค้ำจุนกาย
2. **ผัสสาหาร** — ผัสสะ (การกระทบอารมณ์) เป็นปัจจัยแก่เวทนา
3. **มโนสัญเจตนาหาร** — ความจงใจ/เจตนา เป็นปัจจัยให้เกิดภพใหม่ (กรรม)
4. **วิญญาณาหาร** — วิญญาณ เป็นปัจจัยแก่นามรูป

## ความเชื่อมโยง
อาหาร 4 **มีตัณหาเป็นเหตุ** → สาวย้อนเป็น[[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]]: อาหาร ← ตัณหา ← เวทนา ← ผัสสะ ← สฬายตนะ ← นามรูป ← วิญญาณ ← สังขาร ← อวิชชา

## ความสำคัญ
- เป็นมุมมอง "เหตุปัจจัยของการดำรงอยู่" ที่นำเข้าสู่ปฏิจจสมุปบาท
- ดับตัณหา = ดับรากของอาหาร → ดับภพ

# เสขปฏิปทาสูตร (มัชฌิมนิกาย มัชฌิมปัณณาสก์ สูตรที่ ๕๓ — ข้อปฏิบัติของพระเสขะ)

ที่[[nigrodharama (นิโครธาราม)|นิโครธาราม]] เมือง[[kapilavatthu (กบิลพัสดุ์)|กบิลพัสดุ์]] — เจ้าศากยะเชิญ[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ขึ้นสัณฐาคาร (หอประชุม) ใหม่เป็นปฐมฤกษ์ พระองค์ทรงแสดงธรรมแล้วให้[[ananda (พระอานนท์)|พระอานนท์]]แสดง **เสขปฏิปทา** (ทางของผู้ยังศึกษา) แก่[[mahanama-sakka (เจ้ามหานามศากยะ)|เจ้ามหานามะ]]แทน (ทรงเมื่อยพระปฤษฎางค์)

## องค์ประกอบของเสขปฏิปทา
**ฐานปฏิบัติ:**
1. ถึงพร้อมด้วย**ศีล** (สำรวมในปาติโมกข์)
2. **คุ้มครองทวารอินทรีย์** (ไม่ถือนิมิต/อนุพยัญชนะ)
3. **รู้ประมาณในโภชนะ** (พิจารณาก่อนฉัน)
4. **ชาคริยานุโยค** (ความเพียรเครื่องตื่น)

**[[sappurisa-dhamma-7 (สัปปุริสธรรม 7)|สัปปุริสธรรม 7]]** (ทรัพย์ภายในของสัตบุรุษ):
ศรัทธา / หิริ / โอตตัปปะ / พหูสูต / วิริยะ / สติ / ปัญญา

**[[jhana-4 (ฌาน 4)|ฌาน 4]]** — เครื่องอยู่เป็นสุขในปัจจุบัน

## อุปมาลูกไก่เจาะเปลือกไข่
ผู้พรั่งพร้อมองค์เหล่านี้ = "ฟองไก่ที่แม่ไก่กกอบอุ่นดีแล้ว" → **ควรแก่การเจาะกระเปาะออก** (ตรัสรู้) แม้แม่ไก่ไม่ต้องอธิษฐาน ลูกไก่ก็ออกเองได้
จากจตุตถฌาน → **วิชชา 3** (ความแตกฉาน 3 ขั้น เหมือนลูกไก่เจาะเปลือก): ปุพเพนิวาส → จุตูปปาต → อาสวักขย

## วิชชา-จรณะ
- **จรณะ** (ความประพฤติ) = ศีล + อินทรียสังวร + โภชเนมัตตัญญุตา + ชาคริยะ + สัปปุริสธรรม 7 + ฌาน 4
- **วิชชา** (ความรู้) = วิชชา 3
- ผู้ถึงพร้อมทั้งสอง = **[[vijja-carana (วิชชาและจรณะ)|วิชชาจรณสัมปันโน]]** — ยกคาถาสนังกุมารพรหม: "ผู้ถึงพร้อมด้วยวิชชาและจรณะ ประเสริฐสุดในหมู่เทวดาและมนุษย์"

## ความสำคัญ
- เป็นสูตร**แม่บทของ "เสขปฏิปทา"** — รวมจรณะ 15 (โดยนัย) เป็นโครงปฏิบัติของผู้ยังไม่ถึงอรหัตต์
- พระพุทธเจ้าทรงรับรองคำแสดงของพระอานนท์ ("สาธุ ๆ")

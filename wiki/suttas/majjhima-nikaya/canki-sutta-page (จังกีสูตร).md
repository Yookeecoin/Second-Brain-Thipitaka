# จังกีสูตร (Caṅkī Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ตอบ**กาปทิกมาณพ** (ศิษย์ของ[[canki (จังกีพราหมณ์)|จังกีพราหมณ์]]) ที่[[apana (อาปณนิคม)|อาปณนิคม]] แคว้นโกศล — สูตรว่าด้วย**วิธีเข้าถึงสัจจะ** (ญาณวิทยาทางพุทธ)

- กาปทิกะอ้างว่าพราหมณ์ยึดมนต์เก่าว่า "สิ่งนี้จริง สิ่งอื่นเปล่า" → ทรงชี้ว่าไม่มีพราหมณ์/ฤๅษีองค์ใดรู้เห็นเอง = **"แถวคนตาบอด" (อันธเวณี)** จูงต่อกันโดยไม่มีใครเห็น
- **ธรรม 5 มีวิบาก 2 ส่วน** (อาจจริงหรือเท็จก็ได้): ศรัทธา · รุจิ (ชอบใจ) · อนุสสวะ (ฟังตามกัน) · อาการปริวิตักกะ (ตรึกตามเหตุ) · ทิฏฐินิชฌานขันติ (ทนเพ่งด้วยทิฏฐิ) → **ผู้รักษาสัจจะไม่ด่วนสรุปว่า "นี่จริงเดียว"**
- **[[sacca-3 (ลำดับเข้าถึงสัจจะ)|3 ขั้นเข้าถึงสัจจะ]]**:
  1. **สัจจานุรักษ์** — รักษาสัจจะ: พูดตามจริงว่า "ศรัทธาของเราเป็นอย่างนี้" ไม่ยืนยันว่าจริงเดียว
  2. **สัจจานุโพธ** — ตรัสรู้สัจจะ: ตรวจสอบครู (ปลอดราคะ-โทสะ-โมหะไหม)→ศรัทธา→เข้าไปหา-นั่งใกล้-ฟัง-ทรงจำ-พิจารณา→เพียร→แทงตลอดปรมัตถสัจจะ
  3. **สัจจานุปัตติ** — บรรลุสัจจะ: เสพ-เจริญ-ทำให้มากซึ่งธรรมนั้น
- **ลำดับธรรมอุปการะ** (สืบเหตุถอยหลัง): บรรลุสัจจะ ← ความเพียร ← ปัญญาพิจารณา ← อุตสาหะ ← ฉันทะ ← ธรรมควรเพ่ง ← ใคร่ครวญเนื้อความ ← ทรงจำ ← ฟังธรรม ← เงี่ยโสต ← นั่งใกล้ ← เข้าไปหา ← **ศรัทธา**

# สุนักขัตตสูตร (Sunakkhatta Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ตรัสแก่[[sunakkhatta (สุนักขัตตลิจฉวี)|สุนักขัตตลิจฉวี]] ที่กูฏาคารศาลา ป่ามหาวัน เมือง[[vesali (เวสาลี)|เวสาลี]] — ว่าด้วยผู้**สำคัญผิดว่าตนบรรลุอรหัต** (อธิมานะ)

- ภิกษุหลายรูปพยากรณ์อรหัตผล สุนักขัตตะถามว่าจริงทุกรูปไหม → ทรงตอบ: บางรูปจริง บางรูป**สำคัญผิด**
- **5 ระดับที่จิตน้อมไป** (แต่ละขั้นพรากจากขั้นต่ำกว่า): โลกามิส ([[kamaguna-5 (กามคุณ 5)|กามคุณ 5]]) → อาเนญชสมาบัติ (รูปฌาน-อรูป) → อากิญจัญญายตนะ → เนวสัญญานาสัญญายตนะ → **นิพพานโดยชอบ** — อุปมาใบไม้เหลืองหลุดขั้ว / หินแตกเชื่อมไม่ได้ / ตาลยอดด้วน
- ผู้น้อมใจนิพพานแต่ยังประกอบ**อารมณ์ไม่สบาย (อสัปปายะ)** → ราคะอาจกำเริบ "ตายหรือทุกข์ปางตาย"
- **อุปมาลูกศรอาบยาพิษ**: ลูกศร = [[tanha (ตัณหา)|ตัณหา]], พิษ = [[avijja (อวิชชา)|อวิชชา]] — ถอนแล้วต้องดูแลแผล (ระวังอารมณ์/อาหารแสลง) ไม่งั้นกำเริบ → ผู้สำคัญผิดว่าได้อรหัตย่อมถดถอย

# สังคารวสูตร (Saṅgārava Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ตอบ[[sangarava (สคารวมาณพ)|สคารวมาณพ]] ที่บ้านปัจจลกัปปะ แคว้น[[kosala (โกศล)|โกศล]] — จบมัชฌิมปัณณาสก์ (สูตรที่ 100) ปรารภนางธนัญชานีพราหมณีเปล่ง "นโม" 3 ครั้ง

- สคารวะถาม: สมณพราหมณ์ที่อ้างบรรลุบารมีสูงสุดในปัจจุบัน พระโคดมเป็นพวกไหน? → ทรงจำแนก **3 พวก**:
  1. **ฟังตามกัน** (เตวิชชพราหมณ์ — อนุสสวะ)
  2. **เพียงเชื่อ/ตรึกเอา** (นักตรึก — ตักกี)
  3. **รู้ธรรมเองในธรรมที่ไม่เคยฟัง** (อภิญญา) → พระองค์เป็นพวกนี้
- ทรงเล่า**อัตชีวประวัติการตรัสรู้** (ซ้ำกับ[[pasarasi-sutta-page (ปาสราสิสูตร)|ปาสราสิ]]/[[mahasaccaka-sutta-page (มหาสัจจกสูตร)|มหาสัจจก]]/[[bodhirajakumara-sutta-page (โพธิราชกุมารสูตร)|โพธิราชกุมาร]]): บวช → [[alara-kalama (อาฬารดาบส กาลามโคตร)|อาฬาร]] → [[uddaka-ramaputta (อุททกดาบส รามบุตร)|อุททก]] → **อุปมาไม้ 3 ชนิด** (ไม้สดแช่น้ำ/ไม้สดบนบก/ไม้แห้ง = ผู้ยังติดกาม vs สิ้นกาม) → ทุกรกิริยา (กดฟัน/กลั้นลม/อดอาหาร) → เลิกทรมาน → [[jhana-4 (ฌาน 4)|ฌาน]]-[[vijja-3 (วิชชา 3)|วิชชา 3]]
- ความสำคัญ: ชี้ว่าพระพุทธเจ้าไม่ได้รู้โดยฟังตามหรือคิดเอา แต่**รู้แจ้งด้วยพระองค์เอง**

# จูฬเวทัลลสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๔๔ — ปุจฉา-วิสัชนาธรรมลึก ฉบับย่อ)

ที่เวฬุวัน กรุง[[rajagaha (ราชคฤห์)|ราชคฤห์]] [[visakha-upasaka (วิสาขอุบาสก)|วิสาขอุบาสก]] ถามปัญหาธรรมกับ[[dhammadinna (ธรรมทินนาภิกษุณี)|พระธรรมทินนาภิกษุณี]] (อดีตภรรยา) — เป็นชุดถาม-ตอบ (เวทัลละ) ที่พระพุทธเจ้าทรง**รับรองว่าตอบถูกตรงกับที่พระองค์จะตอบ**

## สักกายะ-สมุทัย-นิโรธ-มรรค (อริยสัจในกรอบ "ตัวตน")
- **สักกายะ** (กายของตน) = [[khandha-5 (ขันธ์ 5)|อุปาทานขันธ์ 5]]
- **สักกายสมุทัย** = [[tanha (ตัณหา)|ตัณหา 3]] (กาม/ภว/วิภว)
- **สักกายนิโรธ** = ความดับตัณหาโดยไม่เหลือ
- **ทางสู่สักกายนิโรธ** = [[ariya-magga-8 (อริยมรรคมีองค์ 8)|อริยมรรคมีองค์ 8]]
- **อุปาทาน vs อุปาทานขันธ์**: ไม่ใช่อันเดียวกัน ไม่ใช่คนละอัน — ฉันทราคะในขันธ์ 5 นั่นแหละคืออุปาทาน

## สักกายทิฏฐิ (ความเห็นว่ามีตัวตน)
ปุถุชนเห็นแต่ละขันธ์เป็นตน 4 นัย (เห็นรูปเป็นตน / ตนมีรูป / รูปในตน / ตนในรูป) × 5 ขันธ์ = **20**; [[anatta (อนัตตา)|อริยสาวก]]ไม่เห็นเช่นนั้น สักกายทิฏฐิจึงไม่มี

## มรรค 8 ย่อลงใน[[sikkha-3 (ไตรสิกขา)|ขันธ์ 3]]
- **สีลขันธ์**: สัมมาวาจา-สัมมากัมมันตะ-สัมมาอาชีวะ
- **สมาธิขันธ์**: สัมมาวายามะ-สัมมาสติ-สัมมาสมาธิ
- **ปัญญาขันธ์**: สัมมาทิฏฐิ-สัมมาสังกัปปะ
- (มรรค 8 ย่อลงในขันธ์ 3 ได้ แต่ขันธ์ 3 ย่อลงในมรรค 8 ไม่ได้)
- **สมาธิ** = จิตมีอารมณ์เดียว; นิมิต = [[satipatthana-4 (สติปัฏฐาน 4)|สติปัฏฐาน 4]]; บริขาร = สัมมัปปธาน 4

## สังขาร 3 + [[nirodha-samapatti (นิโรธสมาบัติ)|นิโรธสมาบัติ]]
- **กายสังขาร** = ลมหายใจ; **วจีสังขาร** = วิตกวิจาร; **จิตตสังขาร** = สัญญา+เวทนา
- เข้านิโรธ: วจีสังขารดับก่อน → กาย → จิต; ออก: จิตเกิดก่อน → กาย → วจี
- ออกแล้วผัสสะ 3 กระทบ (สุญญต/อนิมิต/อัปปณิหิต) จิตน้อมสู่วิเวก

## เวทนา 3 + อนุสัย
- สุข/ทุกข์/อทุกขมสุขเวทนา — สุขคงเพราะตั้งอยู่ แปรเป็นทุกข์เพราะเปลี่ยน
- **อนุสัยที่นอนเนื่อง**: ราคานุสัย ← สุขเวทนา; ปฏิฆานุสัย ← ทุกขเวทนา; อวิชชานุสัย ← อทุกขมสุขเวทนา (แต่ไม่ใช่ทุกกรณี — ผู้ได้ฌานละราคะในสุขได้)
- ส่วนเปรียบ: สุขเวทนา↔ราคะ, ทุกข์↔ปฏิฆะ, อทุกขมสุข↔อวิชชา; **วิชชา↔วิมุตติ↔[[nibbana (นิพพาน)|นิพพาน]]** — "ถามเลยนิพพานไม่ได้ เพราะพรหมจรรย์หยั่งลงในนิพพานเป็นที่สุด"

## ความสำคัญ
- เป็นหลักฐานสำคัญว่า**ภิกษุณีแสดงธรรมลึกได้เทียบเท่าพระเถระ** — พระพุทธเจ้าทรงยกย่องว่าธรรมทินนาเป็นบัณฑิตมีปัญญามาก
- กรอบ "สักกายะ-สมุทัย-นิโรธ-มรรค" = อริยสัจ 4 ในมุมตัวตน
- คู่กับ[[mahavedalla-sutta-page (มหาเวทัลลสูตร)|มหาเวทัลลสูตร]] (สูตรที่ 43 — สารีบุตรตอบมหาโกฏฐิกะ)

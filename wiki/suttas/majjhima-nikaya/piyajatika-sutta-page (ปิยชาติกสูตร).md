# ปิยชาติกสูตร (Piyajātika Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ตรัสหลักธรรม "**โสกะ ปริเทวะ ทุกข์ โทมนัส อุปายาส ย่อมเกิดแต่ของที่รัก** (ปิยชาติ)" ที่[[jetavana (เชตวัน)|เชตวัน]] เมือง[[savatthi (สาวัตถี)|สาวัตถี]]

- คฤหบดีเสียบุตรคนเดียว เศร้าจนเพ้อ ไปเฝ้า → ทรงตรัสว่าทุกข์เกิดแต่ของรัก; คฤหบดี**ไม่ยอมรับ** (เถียงว่าของรักนำมาซึ่งความสุข-โสมนัสต่างหาก) แล้วไปคุยกับนักเลงสะกาที่เห็นพ้องด้วย
- เรื่องลือถึงวัง → พระเจ้า[[pasenadi (พระเจ้าปเสนทิโกศล)|ปเสนทิ]]ก็ไม่เชื่อ; [[mallika (พระนางมัลลิกา)|พระนางมัลลิกา]]ส่งนาฬิชังฆพราหมณ์ไปทูลถามยืนยันพุทธพจน์ (ทรงยกตัวอย่างคนเป็นบ้า/ฆ่าตัวตายเพราะสูญเสียคนรัก)
- **อุบายของมัลลิกา**: ถามพระเจ้าปเสนทิทีละข้อว่า วชิรีกุมารี / วาสภขัตติยา / วิฑูฑภเสนาบดี / ตัวมัลลิกาเอง / แคว้นกาสี-โกศล เป็นที่รักไหม — ถ้าสิ่งเหล่านั้นแปรปรวนจะเกิดโสกะ-ปริเทวะไหม → ปเสนทิยอมรับทุกข้อ จึงเข้าใจและนอบน้อมพระพุทธเจ้า 3 ครั้ง
- ใจความ: **ความรัก/ผูกพันเป็นบ่อเกิดของทุกข์** (โยง[[tanha (ตัณหา)|ตัณหา]], [[dukkha (ทุกข์)]]) — ยิ่งรักมากยิ่งทุกข์มากเมื่อพลัดพราก

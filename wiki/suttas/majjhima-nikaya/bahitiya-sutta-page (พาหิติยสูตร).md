# พาหิติยสูตร (Bāhitika Sutta)

[[ananda (พระอานนท์)|พระอานนท์]]ตอบปัญหาพระเจ้า[[pasenadi (พระเจ้าปเสนทิโกศล)|ปเสนทิ]] ที่ฝั่งแม่น้ำอจิรวดี (ใกล้บุพพาราม เมือง[[savatthi (สาวัตถี)|สาวัตถี]]) — ชื่อสูตรจากผ้า "พาหิติกา" ที่ถวายท้ายเรื่อง

- ปเสนทิถาม: [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ประพฤติ**กาย-วาจา-มโนสมาจาร**ที่วิญญูชนติเตียนได้บ้างไหม → พระอานนท์ตอบ **ไม่เลย** (ตถาคตละอกุศลทั้งปวง ประกอบกุศลทั้งปวง)
- **นิยามสมาจารที่พึงติเตียน** (ไล่เป็นลำดับ): สมาจารอกุศล = มีโทษ = มีความเบียดเบียน = **มีทุกข์เป็นวิบาก** = เป็นไปเพื่อเบียดเบียนตน/ผู้อื่น/ทั้งสอง ทำให้**อกุศลเจริญ กุศลเสื่อม**
- ฝ่ายตรงข้าม = สมาจารกุศล: ไม่มีโทษ ไม่เบียดเบียน มีสุขเป็นวิบาก ทำให้กุศลเจริญ อกุศลเสื่อม — วิญญูชนไม่ติเตียน
- เกณฑ์ตัดสินดี-ชั่ว = **ผล (วิบาก) + การเบียดเบียน** ไม่ใช่รูปแบบภายนอก
- ปเสนทิเลื่อมใส ถวาย**ผ้าพาหิติกา** (ผ้าที่[[ajatasattu (พระเจ้าอชาตศัตรู)|พระเจ้าอชาตศัตรู]]ถวายมา) แก่พระอานนท์ → พระอานนท์นำไปถวายพระพุทธเจ้า; ทรงสรรเสริญว่า "เป็นลาภของปเสนทิที่ได้พบและนั่งใกล้อานนท์"

# มหาโคสิงคสาลสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๓๒ — "อุปมาป่าโคสิงค" ฉบับใหญ่)

ที่**ป่าโคสิงคสาลวัน** [[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ประทับพร้อมพระเถระผู้มีชื่อเสียงหลายรูป คืนหนึ่งราตรีแจ่มกระจ่าง ไม้สาละบานสะพรั่ง กลิ่นคล้ายทิพย์ฟุ้งไป — [[sariputta (พระสารีบุตร)|พระสารีบุตร]]ตั้งคำถามวงสนทนาว่า **"ป่าโคสิงคสาลวันพึงงามด้วยภิกษุเช่นไร?"**

## คำตอบของพระเถระแต่ละรูป (ตามความถนัด)

| พระเถระ | ภิกษุที่ทำให้ป่างาม |
|---|---|
| [[ananda (พระอานนท์)\|พระอานนท์]] | ภิกษุ **พหูสูต** ทรงจำธรรม แสดงแก่บริษัท 4 เพื่อถอนอนุสัย |
| [[revata (พระเรวตะ)\|พระเรวตะ]] | ภิกษุยินดีใน **ความหลีกเร้น** เจริญเจโตสมถะ มีฌาน + วิปัสสนา พอกพูนสุญญาคาร |
| [[anuruddha (พระอนุรุทธะ)\|พระอนุรุทธะ]] | ภิกษุมี **ทิพยจักษุ** ตรวจดูโลกพันหนึ่งได้ |
| [[mahakassapa (พระมหากัสสปะ)\|พระมหากัสสปะ]] | ภิกษุ **ถือธุดงค์** (อยู่ป่า/บิณฑบาต/บังสุกุล/ไตรจีวร) มักน้อย สันโดษ และสรรเสริญคุณนั้นแก่ผู้อื่น |
| [[moggallana (พระมหาโมคคัลลานะ)\|พระมหาโมคคัลลานะ]] | ภิกษุ 2 รูป **สนทนาอภิธรรม** ถาม-แก้กันไม่ติดขัด |
| [[sariputta (พระสารีบุตร)\|พระสารีบุตร]] | ภิกษุ **กำกับจิตได้** (ไม่ตกอยู่ใต้อำนาจจิต) เข้าวิหารสมาบัติใดเมื่อใดก็ได้ดังใจ — อุปมาหีบผ้าของพระราชาเลือกห่มได้ทุกเวลา |

## ทูลถามพระพุทธเจ้า — "คำของใครเป็นสุภาษิต?"
ทั้งหมดไปเฝ้าพระพุทธเจ้า พระองค์ตรัสว่า **"คำของพวกเธอทั้งหมดเป็นสุภาษิตโดยปริยาย"** (รับรองทุกคำตามความถนัดของแต่ละรูป) แล้วทรงเพิ่มคำตอบของพระองค์เอง:

> ภิกษุกลับจากบิณฑบาต นั่งคู้บัลลังก์ ตั้งกายตรง ตั้งสติมั่น **อธิษฐานว่า "จิตของเรายังไม่หลุดพ้นจากอาสวะเพียงใด เราจักไม่ทำลายบัลลังก์นี้เพียงนั้น"** — ภิกษุเช่นนี้แหละทำให้ป่าโคสิงคสาลวันงาม

## ความสำคัญ
- เป็น "**แผนที่คุณสมบัติ**" ของสาวกชั้นเลิศ 6 ด้าน (พหูสูต / วิเวก / ทิพยจักษุ / ธุดงค์ / ปัญญาแตกฉาน / วสีในสมาบัติ) — แต่ละด้านล้วนงาม
- พระพุทธเจ้าทรงปิดท้ายด้วย **ความเพียรเด็ดเดี่ยวเพื่อสิ้นอาสวะ** = ยอดเหนือทุกคุณสมบัติ
- คู่กับ[[culagosinga-sutta-page (จูฬโคสิงคสาลสูตร)|จูฬโคสิงคสาลสูตร]] (สูตรที่ 31 — ป่าเดียวกัน เน้นสามัคคี)

# โพธิราชกุมารสูตร (Bodhirājakumāra Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ตรัสแก่**โพธิราชกุมาร** ที่โกกนุทปราสาท เภสกฬาวัน เมือง[[sumsumaragira (สุงสุมารคิระ)|สุงสุมารคิระ]] แคว้นภัคคะ

- เปิดเรื่อง: โพธิราชกุมารให้ปูผ้าขาวรับเสด็จ ทรง**ไม่เหยียบ** ([[ananda (พระอานนท์)|พระอานนท์]]ทูลให้เก็บผ้า "ตถาคตทรงเล็งถึงชนรุ่นหลัง" — อรรถกถาว่าราชกุมารไม่มีบุตร)
- ทิฏฐิของโพธิราชกุมาร: "**สุขเข้าถึงไม่ได้ด้วยสุข ต้องเข้าถึงด้วยทุกข์**" → ทรงเล่า**อัตชีวประวัติการแสวงหาโพธิญาณ** (เหมือน[[pasarasi-sutta-page (ปาสราสิสูตร)|ปาสราสิสูตร]] / [[mahasaccaka-sutta-page (มหาสัจจกสูตร)|มหาสัจจกสูตร]]): บวช → [[alara-kalama (อาฬารดาบส กาลามโคตร)|อาฬาร]] (อากิญจัญญายตนะ) → [[uddaka-ramaputta (อุททกดาบส รามบุตร)|อุททก]] (เนวสัญญานาฯ) → ไม่พอใจ → **อุปมาไม้ 3 ชนิด** → ทุกรกิริยา → เลิกทรมานตน → [[jhana-4 (ฌาน 4)|ฌาน]] → [[vijja-3 (วิชชา 3)|วิชชา 3]] → ตรัสรู้ → พรหมอาราธนา → โปรด[[panchavaggiya (ปัญจวัคคีย์)|ปัญจวัคคีย์]]
- คำถามสำคัญ: ภิกษุใช้เวลานานเท่าไรจึงบรรลุ? → ตอบด้วย **[[padhaniyanga-5 (ปธานิยังคะ 5)|ปธานิยังคะ 5]]** (องค์แห่งความเพียร): ศรัทธา · อาพาธน้อย · ไม่โอ้อวด-ซื่อตรง · ปรารภความเพียร · มีปัญญา (เห็นความเกิด-ดับ) — ผู้พร้อมองค์ 5 ครบ "สอนเย็นบรรลุเช้า สอนเช้าบรรลุเย็น"
- ผู้ทูลนิมนต์คือมาณพสัญชิกาบุตร

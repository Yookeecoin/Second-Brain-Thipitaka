# จูฬหัตถิปโทปมสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๒๗ — "อุปมารอยเท้าช้างย่อ")

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ทรงแสดงแก่ [[janussoni (ชาณุสโสณิพราหมณ์)|ชาณุสโสณิพราหมณ์]] ที่[[savatthi (สาวัตถี)|พระเชตวัน เมืองสาวัตถี]] หลังพราหมณ์ได้ฟังปิโลติกปริพาชกชมพระพุทธเจ้า — สูตรนี้วาง **"แผนที่ปฏิบัติ"** จากศีล → สมาธิ → วิชชา 3 อย่างเป็นระบบ

## ฉาก: ปิโลติกปริพาชกชมพระพุทธเจ้า
ปริพาชกบอกชาณุสโสณิว่าเห็น **"ร่องรอย 4 อย่าง"** ในพระสมณโคดม จึงเชื่อว่าเป็นอรหันตสัมมาสัมพุทธะ:
1. ขัตติยบัณฑิตคิดผูกปัญหามาท้าทาย → ได้ฟังธรรม → ไม่ถาม กลายเป็นสาวก
2. พราหมณบัณฑิต — เหมือนกัน
3. คฤหบดีบัณฑิต — เหมือนกัน
4. **สมณบัณฑิต — ขอบวช → บำเพ็ญเพียร → บรรลุพระอรหันต์**

## พระพุทธเจ้าทรงตรวจ "ยังไม่ครบ"
ทรงสอนว่าอุปมา 4 ร่องรอยยังไม่บริบูรณ์ — เหมือนพรานช้างเห็นรอยเท้าใหญ่ ก็ยังตัดสินไม่ได้ว่าเป็นช้างใหญ่ (อาจเป็นพังตัวเตี้ย/พังสูง ที่มีรอยใหญ่ก็ได้) **ต้องเห็นทั้งตัวช้างจริงๆ จึงตัดสินใจได้**

## "อุปมารอยเท้าช้าง" ที่บริบูรณ์ = แผนที่ปฏิบัติ 4 ขั้น
แต่ละขั้นทรงเรียกว่า **"ตถาคตบท / ตถาคตนิเสวิตะ / ตถาคตารัญชิตะ"** (รอยเท้า/ที่ถูกเสียดสี/ที่ถูกขนายแทง ของตถาคต) — อริยสาวกผ่านครบทุกขั้นจึง "ตกลงใจ" ในพระรัตนตรัยจริง

### 1. ออกบวช + ศีลบริบูรณ์
- ฟังธรรม → เห็นว่า "ฆราวาสคับแคบ บรรพชาปลอดโปร่ง"
- ละโภคะ-ญาติ ปลงผมหนวด บวช
- **จุลศีล-มัชฌิมศีล-มหาศีล** (ละปาณาฯ อทินนาฯ เมถุน มุสา ปิสุณา ผรุสา สัมผัปป. รับเงินทอง แพะ-แกะ-ไก่-สุกร-ช้าง-โค ฯลฯ)
- **อินทรียสังวร** (สำรวมตา-หู-จมูก-ลิ้น-กาย-ใจ)
- **สติสัมปชัญญะ** (ในการเดิน ยืน นั่ง นอน กิน ดื่ม)
- **สันโดษ** (ดุจนกมีปีกของตน)

### 2. ละนิวรณ์ + รูปฌาน 4
ละ[[nivarana-5 (นิวรณ์ 5)|นิวรณ์ 5]] → บรรลุปฐมฌาน → ทุติย → ตติย → จตุตถฌาน
แต่ละขั้น **ทรงเรียกว่า "ตถาคตบท"** อริยสาวกยังไม่ตกลงใจ

### 3. ปุพเพนิวาสญาณ (ระลึกชาติ)
ระลึกชาติได้เป็นอันมาก — **"ตถาคตบท"** ยังไม่ตกลงใจ

### 4. จุตูปปาตญาณ (ทิพยจักษุเห็นจุติ-อุบัติ)
เห็นสัตว์เป็นไปตามกรรม — **"ตถาคตบท"** ยังไม่ตกลงใจ

### 5. อาสวักขยญาณ — **เห็นช้างเต็มตัว**
รู้ชัดอริยสัจ 4 + อาสวะ 4 → จิตหลุดพ้น → "ชาติสิ้นแล้ว" → **ตอนนี้แหละจึงตกลงใจในพระรัตนตรัย**

## บทสรุป
ชาณุสโสณิพราหมณ์เลื่อมใส → ถึงพระรัตนตรัยเป็นสรณะ เป็นอุบาสกตลอดชีวิต

## ความสำคัญ
เป็นแผนที่ "ปฏิบัติเข้าทาง" ครบลำดับขั้น — ก่อน[[sammaditthi-sutta-page (สัมมาทิฏฐิสูตร)|สัมมาทิฏฐิสูตร]]จะวิเคราะห์ทฤษฎี สูตรนี้วางลำดับการลงมือจริง: ศีล → สมาธิ → วิชชา 3 → อาสวักขย ใช้เป็นโครงคำสอนของหลายสูตรในมัชฌิมนิกาย

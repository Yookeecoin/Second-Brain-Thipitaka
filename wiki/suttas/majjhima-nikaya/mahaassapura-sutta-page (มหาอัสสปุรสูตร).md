# มหาอัสสปุรสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๓๙ — ทางปฏิบัติของ "สมณะ" ฉบับใหญ่)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ทรงแสดงที่อัสสปุรนิคม แคว้น[[anga (อังคะ)|อังคะ]] — คนเรียกพวกเธอว่า "สมณะ" พวกเธอก็รับว่าเป็นสมณะ จึงต้อง **ปฏิบัติให้สมชื่อ** เพื่อให้คำปฏิญญาเป็นจริง และทานของผู้ถวายมีผลมาก

## ลำดับธรรมที่ทำให้ "เป็นสมณะจริง" (ไล่ขั้นไม่ให้พอใจกลางทาง)
ทุกขั้น พระพุทธเจ้าเตือนว่า "อย่าเพิ่งพอใจว่าทำเสร็จแล้ว เมื่อยังมีกิจสูงกว่า"
1. **หิริโอตตัปปะ** (ละอาย-เกรงบาป)
2. **กายสมาจาร–วจีสมาจาร–มโนสมาจาร–อาชีวะ บริสุทธิ์** (เปิดเผย ไม่ยกตนข่มผู้อื่น)
3. **อินทรียสังวร** (สำรวมอินทรีย์ 6 ไม่ถือนิมิต/อนุพยัญชนะ)
4. **โภชเนมัตตัญญุตา** (รู้ประมาณในอาหาร พิจารณาก่อนฉัน)
5. **ชาคริยานุโยค** (ความเพียรเครื่องตื่น — จงกรม/นั่งชำระนิวรณ์ นอนอย่างราชสีห์)
6. **สติสัมปชัญญะ** (รู้ตัวในอิริยาบถทั้งปวง)
7. เสพเสนาสนะสงัด → **ละ[[nivarana-5 (นิวรณ์ 5)|นิวรณ์ 5]]**

## อุปมานิวรณ์ 5 (ก่อนเข้าฌาน)
นิวรณ์ที่ยังละไม่ได้ = หนี้ / โรค / เรือนจำ / ความเป็นทาส / ทางไกลกันดาร; เมื่อละได้ = ปลดหนี้ / หายโรค / พ้นคุก / เป็นไท / ถึงที่เกษม → เกิด**ปราโมทย์-ปีติ**

## [[jhana-4 (ฌาน 4)|ฌาน 4]] (พร้อมอุปมา)
- ปฐมฌาน — ดุจก้อนผงสบู่ชุ่มน้ำทั่วทั้งก้อน
- ทุติยฌาน — ดุจห้วงน้ำมีตาน้ำผุดเย็นทั่ว
- ตติยฌาน — ดุจดอกบัวที่น้ำหล่อเลี้ยงทั้งกอ
- จตุตถฌาน — ดุจคนคลุมผ้าขาวทั่วตัว

## [[vijja-3 (วิชชา 3)|วิชชา 3]] → สิ้นอาสวะ
ปุพเพนิวาสานุสติ → จุตูปปาตญาณ → อาสวักขยญาณ (รู้[[ariyasacca-4 (อริยสัจ 4)|อริยสัจ 4]]) → จิตหลุดพ้น (อุปมาเห็นปลา-กรวดในห้วงน้ำใส)

## นิยาม 7 ชื่อของผู้ละกิเลสได้
ภิกษุผู้ถึงที่สุดเรียกได้ว่า **สมณะ** (ระงับบาป) / **พราหมณ์** (ลอยบาป) / **นหาตกะ** (อาบล้างบาป) / **เวทคู** (รู้แจ้ง) / **โสตติยะ** (ทำบาปให้หลับ) / **อริยะ** (ไกลจากบาป) / **อรหันต์** (กำจัดบาป)

## ความสำคัญ
- เป็น "**คู่มือลำดับปฏิบัติของภิกษุ**" ที่ครบสมบูรณ์ที่สุดสูตรหนึ่ง = โครง[[sikkha-3 (ไตรสิกขา)|ไตรสิกขา]]ขยายเต็ม (ศีล-อินทรียสังวร-โภชน์-ชาคริยะ-สติ → สมาธิ(ฌาน) → ปัญญา(วิชชา 3))
- คู่กับ[[culaassapura-sutta-page (จูฬอัสสปุรสูตร)|จูฬอัสสปุรสูตร]] (สูตรที่ 40)

# สังขารูปปัตติสูตร (Saṅkhārupapatti Sutta)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ทรงแสดงแก่ภิกษุทั้งหลาย ณ [[jetavana (เชตวัน)|เชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]] — ว่าด้วย "ความเกิดขึ้นแห่งสังขาร" คือ ความปรารถนา (เจตนา/อธิษฐาน) ประกอบกับศรัทธา ศีล สุตะ จาคะ ปัญญา ย่อมนำไปเกิดในภพที่ปรารถนานั้น

## หลักการ
ภิกษุผู้ประกอบด้วยคุณสมบัติ 5 ประการ คือ
- **ศรัทธา** — เชื่อความตรัสรู้ของพระพุทธเจ้า
- **ศีล** — รักษาศีลบริสุทธิ์
- **สุตะ** — ได้ยินได้ฟังธรรมมาก
- **จาคะ** — มีใจเอื้อเฟื้อ สละ
- **ปัญญา** — มีปัญญาหยั่งรู้

เมื่อมีความปรารถนาและตั้งจิต อธิษฐานจิต เจริญจิตนั้น ย่อมสำเร็จตามปรารถนา

## ลำดับภพที่ปรารถนาได้ (ข้อ 319–332)
- กษัตริย์มหาศาล / พราหมณ์มหาศาล / คฤหบดีมหาศาล
- เทวดาชั้นดาวดึงส์ / ยามา / ดุสิต / นิมมานรดี / ปรนิมมิตวสวัตดี
- สหัสสพรหม / ทวิสหัสสพรหม / ติสหัสสพรหม / จตุสหัสสพรหม / ปัญจสหัสสพรหม / ทสสหัสสพรหม / สตสหัสสพรหม
- เทวดาชั้นอาภา / ปริตตาภา / อัปปมาณาภา / อาภัสสรา
- เทวดาชั้นสุภา / ปริตตสุภา / อัปปมาณสุภา / สุภกิณหา
- เทวดาชั้นเวหัปผลา / อวิหา / อตัปปา / สุทัสสา / สุทัสสี / อกนิฏฐา
- เทวดาผู้เข้าถึงอากาสานัญจายตนภพ / วิญญาณัญจายตนภพ / อากิญจัญญายตนภพ / เนวสัญญานาสัญญายตนภพ
- **บรรลุเจโตวิมุตติ ปัญญาวิมุตติ อาสวักขัย** — ภิกษุนั้นย่อมไม่เกิดในที่ไหนๆ อีก (จุดหมายสูงสุด)

## ความสำคัญ
- แสดงว่า "สังขาร" (ความตั้งใจ เจตนา อธิษฐาน) คือปัจจัยกำหนดภพหน้า ไม่ใช่โชคหรือพรสวรรค์
- คุณสมบัติ 5 ข้อ (ศรัทธา ศีล สุตะ จาคะ ปัญญา) เป็น "ทุนปฏิบัติ" ที่รองรับทุกความปรารถนา ทั้งโลกิยะและโลกุตตระ
- ปิดอนุปทวรรคด้วยสูตรที่เชื่อมโยงการปฏิบัติกับผลในวัฏสงสาร จนถึงที่สุดคือวิมุตติ
- โยง: [[ariya-magga-8 (อริยมรรคมีองค์ 8)]], [[saddha (ศรัทธา)]], [[panna (ปัญญา)]]

# จูฬโคปาลสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๓๔ — "อุปมานายโคบาล" ฉบับย่อ)

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]]ทรงแสดงที่ฝั่งแม่น้ำคงคา จังหวัดอุกกเวลา แคว้น[[vajji (วัชชี)|วัชชี]] — อุปมา **นายโคบาลพาฝูงโคข้ามแม่น้ำคงคา** ไปฝั่งวิเทหรัฐ

## นายโคบาล 2 แบบ
- **โง่** — ไม่ดูฤดู ไม่ตรวจท่า ต้อนฝูงข้ามผิดท่า → ฝูงโควนเวียนกลางกระแส **วอดวาย** = สมณพราหมณ์ที่ไม่ฉลาดต่อโลกนี้/โลกหน้า/แดนมาร/แดนมฤตยู → ผู้เชื่อตามย่อมพินาศ
- **ฉลาด** — ดูฤดู ตรวจท่า ต้อนข้ามถูกท่า → ฝูงถึงฝั่งสวัสดี = สมณพราหมณ์ผู้ฉลาดต่อโลกทั้งสอง → ผู้เชื่อตามย่อมได้ประโยชน์สุข

## ลำดับโคข้ามน้ำ = อริยบุคคล "ตัดกระแสมาร"
| โคในอุปมา | อริยบุคคล |
|---|---|
| โคอุสภะ (จ่าฝูง) ข้ามก่อน | **พระอรหันต์** ขีณาสพ |
| โคใช้งานได้/ฝึกได้ | **พระอนาคามี** (สิ้นสังโยชน์เบื้องต่ำ 5 ผุดเกิดในสุทธาวาส) |
| โคหนุ่มสาวคะนอง | **พระสกทาคามี** (สิ้นสังโยชน์ 3 + ราคะ-โทสะ-โมหะเบาบาง) |
| ลูกโค/โคผอม | **พระโสดาบัน** (สิ้นสังโยชน์ 3 ไม่ตกต่ำ เที่ยงตรัสรู้) |
| ลูกโคอ่อนเกิดวันนั้น ว่ายตามเสียงแม่ | ภิกษุผู้**ตามธรรมและศรัทธา** (ธัมมานุสารี/สัทธานุสารี) |

ทั้งหมด **ตัดตรงกระแสมาร ถึงฝั่งโดยสวัสดี** — พระพุทธเจ้าตรัสว่าพระองค์เองคือผู้ฉลาดทั้งโลกนี้โลกหน้า เปิดประตูอมตะ (พระนิพพาน) ให้

## ความสำคัญ
- เทียบลำดับ[[ariya-puggala-4 (อริยบุคคล 4)|อริยบุคคล 4]] + อนุสารี 2 เป็นภาพ "ข้ามแม่น้ำ" ที่จำง่าย
- ต่างจาก[[mahagopalaka-sutta-page (มหาโคปาลสูตร)|มหาโคปาลสูตร]] (สูตรที่ 33) ที่เน้นองค์ 11 ของภิกษุ ส่วนสูตรนี้เน้น "ข้ามกระแสมาร" และการเลือกครูที่ฉลาด

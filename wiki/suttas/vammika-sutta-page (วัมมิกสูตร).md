# วัมมิกสูตร (มัชฌิมนิกาย มูลปัณณาสก์ สูตรที่ ๒๓ — "อุปมาจอมปลวก")

[[buddha (พระพุทธเจ้า)|พระพุทธเจ้า]] ตรัสตอบปริศนาที่เทวดามาบอก [[kumara-kassapa (พระกุมารกัสสป)|พระกุมารกัสสป]] ที่ป่าอันธวัน เมือง[[savatthi (สาวัตถี)|สาวัตถี]] — เป็น "อุปมานิทาน 15 ข้อ" ที่เทวดาท้าให้ขุดจอมปลวก

## ปริศนาที่เทวดาถาม
เทวดาบอกพระกุมารกัสสปว่า "จอมปลวกพ่นควันกลางคืน ลุกโพลงกลางวัน พราหมณ์บอกสุเมธะให้ขุด — ขุดทีละชั้นเจอของ 8 อย่าง" แล้วให้ไปทูลถามพระศาสดา (ใครที่อื่นตอบไม่ได้ นอกจากตถาคต)

## ปริศนา 15 ข้อ + คำตอบ
| ปริศนา | ความหมาย |
|---|---|
| **จอมปลวก** | กายอันเกิดจากมหาภูตรูป 4 (มีบิดามารดาเป็นแดนเกิด, ไม่เที่ยง) |
| **พ่นควันกลางคืน** | กลางวันขมักเขม้นทำงาน → กลางคืนตรึกตรองถึง |
| **ลุกโพลงกลางวัน** | กลางคืนตรึก → กลางวันลงมือทำกาย-วาจา |
| **พราหมณ์** | พระตถาคต |
| **สุเมธะ** | เสขภิกษุ (ผู้ปฏิบัติ) |
| **ศาสตรา** | ปัญญาอันประเสริฐ |
| **ขุด** | ปรารภความเพียร |
| **ลิ่มสลัก** | อวิชชา (ละ-ขุด) |
| **อึ่ง** | ความคับแค้นเพราะโกรธ (ละ-ขุด) |
| **ทาง 2 แพร่ง** | วิจิกิจฉา (ละ-ก่น) |
| **หม้อกรองน้ำด่าง** | [[nivarana-5 (นิวรณ์ 5)|นิวรณ์ 5]] (ละ-ยก) |
| **เต่า** | อุปาทานขันธ์ 5 (ละ-ยก) |
| **เขียงหั่นเนื้อ** | [[kamaguna-5 (กามคุณ 5)|กามคุณ 5]] (ละ-ยก) |
| **ชิ้นเนื้อ** | นันทิราคะ (ละ-ยก) |
| **นาค** | **ภิกษุผู้ขีณาสพ (พระอรหันต์)** — "นาคจงอยู่เถิด อย่าเบียดเบียน จงนอบน้อม" |

## ความสำคัญ
สูตรนี้มีโครงเป็น "เกมปริศนา" 15 ข้อ ที่ค่อยๆ ไล่จากกายสังขาร → กิเลสหยาบ-ละเอียด → จบที่พระอรหันต์ เป็นแผนที่ปฏิบัติย่อยๆ ลำดับขั้น ใครเข้าใจครบ 15 ข้อ = เข้าใจทั้งคำสอน

## ที่อ้างอิง
มมร เล่ม 18 (มัชฌิมนิกาย มูลปัณณาสก์ โอปัมมวรรค) ข้อ ๒๘๙–๒๙๑ | ม.มู. ๑๒

# สฬายตนสังยุต — คหปติวรรค (วรรคที่ 3, ตติยปัณณาสก์)

วรรคที่ 3 ของ**ตติยปัณณาสก์** ใน[[salayatana-suddhavagga-page (สฬายตนสังยุต-สุทธวรรค)|สฬายตนสังยุต]] — 10 สูตร ผู้ฟังส่วนใหญ่เป็นคหบดี (คฤหบดี/คฤหัสถ์) และพระเจ้าแผ่นดิน

- **เวสาลีสูตร (ข้อ ๑๙๑–๑๙๒)** — [[ugga-vesali (อุคคคฤหบดีชาวเวสาลี)|อุคคคฤหบดีชาวเวสาลี]]เข้าเฝ้าที่[[vesali (กรุงเวสาลี)|กรุงเวสาลี]] กูฏาคารศาลา ป่ามหาวัน → ทรงตรัส: เหตุที่ปรินิพพาน/ไม่ปรินิพพาน = ไม่เพลิดเพลิน/เพลิดเพลินในอายตนะ ๖ → วิญญาณอาศัยตัณหา
- **วัชชีสูตร (ข้อ ๑๙๓–๑๙๔)** — [[ugga-hatthi (อุคคคฤหบดีชาวหัตถิคาม)|อุคคคฤหบดีชาวหัตถิคาม แคว้นวัชชี]] → เหมือนเวสาลีสูตร
- **นาฬันทสูตร (ข้อ ๑๙๔)** — [[upali-gahapati (อุบาลีคฤหบดี)|อุบาลีคฤหบดี]]ที่**ปาวาริกัมพวัน เมืองนาฬันทา** → เหมือนสูตรก่อน (*หมายเหตุ: ข้อ ๑๙๔ ปรากฏซ้ำทั้งท้ายวัชชีสูตรและต้นนาฬันทสูตร อาจเป็นข้อต่อเนื่องหรือเลขซ้ำในต้นฉบับ*)
- **ภารทวาชสูตร (ข้อ ๑๙๕–๑๙๘)** — [[pindola-bharadvaja (พระบิณโฑลภารทวาชะ)|พระบิณโฑลภารทวาชะ]]อยู่ที่**[[kosambi (โกสัมพี)|กรุงโกสัมพี]] โฆสิตาราม** → [[udena (พระเจ้าอุเทน)|พระเจ้าอุเทน]]ถาม: ภิกษุหนุ่มถือพรหมจรรย์ได้อย่างไร? → ท่านตอบ **3 เหตุ**: (1) เห็นภิกษุณีสูงวัยด้วย **มาตาทิสัญญา** (เหมือนมารดา-พี่สาว-น้องสาว) (2) เจริญ **อสุภ** (พิจารณาร่างกายว่าไม่งาม) (3) **อินทรียสังวร** → [[udena (พระเจ้าอุเทน)|พระเจ้าอุเทน]]ถึงไตรสรณะ
- **โสณสูตร (ข้อ ๑๙๙–๒๐๐)** — [[sona-gahapati (โสณะบุตรคฤหบดี)|โสณะบุตรคฤหบดี]]ที่[[rajagaha (ราชคฤห์)|กรุงราชคฤห์]] **เวฬุวัน** → เหมือนเวสาลีสูตร
- **โฆสิตสูตร (ข้อ ๒๐๐)** — [[ghosita-gahapati (โฆสิตคฤหบดี)|โฆสิตคฤหบดี]]ถาม[[ananda (พระอานนท์)|พระอานนท์]]ที่**[[kosambi (โกสัมพี)|กรุงโกสัมพี]] โฆสิตาราม**: "ความแตกต่างแห่งธาตุคืออะไร?" → พระอานนท์ตอบ: **ธาตุ ๖** (จักขุ-โสต-ฆาน-ชิวหา-กาย-มน) + อารมณ์ ๖ → เกิด **เวทนา ๓** (สุข-ทุกข์-อทุกขมสุข) (*หมายเหตุ: ข้อ ๒๐๐ ปรากฏซ้ำทั้งท้ายโสณสูตรและต้นโฆสิตสูตร*)
- **หาลิททกานิสูตร (ข้อ ๒๐๑–๒๐๒)** — [[haliddhakani-gahapati (หาลิททกานิคฤหบดี)|หาลิททกานิคฤหบดี]]ถาม[[maha-kaccayana (พระมหากัจจายนะ)|พระมหากัจจายนะ]]ที่**สังปวัตตบรรพต แคว้นอวันตี** → **ธาตุ → ผัสสะ → เวทนา**: อาศัยธาตุ ๖ เกิดผัสสะ ๖ เกิดเวทนา ๓
- **นกุลปิตุสูตร (ข้อ ๒๐๓–๒๐๔)** — [[nakulapita-gahapati (นกุลบิดาคฤหบดี)|นกุลบิดาคฤหบดี]]ที่**[[sumsumaragiri (สุงสุมารคิรินคร)|สุงสุมารคิรินคร]] เภสกฬาวัน ภัคคชนบท** → เหมือนเวสาลีสูตร
- **โลหิจจสูตร (ข้อ ๒๐๕–๒๐๙)** — [[maha-kaccayana (พระมหากัจจายนะ)|พระมหากัจจายนะ]]อยู่ที่**อรัญญกุฎี ใกล้มักกรกฏนคร แคว้นอวันตี** → มาณพของ[[lohicca (โลหิจจพราหมณ์)|โลหิจจพราหมณ์]]ด่าว่า → ท่านกล่าวคาถา "พราหมณ์เก่า" (อินทรียสังวร-เนกขัมมะ) → [[lohicca (โลหิจจพราหมณ์)|โลหิจจพราหมณ์]]ได้ฟังแล้วถึงไตรสรณะ
- **เวรหัญจานีสูตร (ข้อ ๒๑๐–๒๑๒)** — [[udayi (พระอุทายี)|พระอุทายี]]อยู่ที่**สวนมะม่วงโตเทยยพราหมณ์ กามัณฑานคร** → [[verahaccani (พราหมณีเวรหัญจานิโคตร)|พราหมณีเวรหัญจานิโคตร]]เข้าหา: **ครั้งแรกนั่งบนอาสนะสูงคลุมศีรษะ** ท่านไม่แสดงธรรม; **ครั้งสองนั่งอาสนะต่ำเปิดศีรษะ** ท่านจึงแสดงธรรม "เมื่อมีอายตนะ จึงบัญญัติสุข-ทุกข์ได้" → พราหมณีถึงไตรสรณะ

**ความสำคัญ:** วรรคนี้แสดงว่าคำสอนเรื่องอายตนะถูกถ่ายทอดให้คฤหัสถ์และกษัตริย์ในหลายชนบท — ภูมิภาคหลัก ได้แก่ [[vesali (กรุงเวสาลี)|เวสาลี]] [[kosambi (โกสัมพี)|โกสัมพี]] [[rajagaha (ราชคฤห์)|ราชคฤห์]] และ**แคว้นอวันตี** (แดนไกล) โดย[[maha-kaccayana (พระมหากัจจายนะ)|พระมหากัจจายนะ]]ทำหน้าที่แทนพระพุทธองค์ในชนบทห่างไกล

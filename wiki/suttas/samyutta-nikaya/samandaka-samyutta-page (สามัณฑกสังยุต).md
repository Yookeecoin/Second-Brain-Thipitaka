# สามัณฑกสังยุต

**สามัณฑกปริพาชก** ถาม [[sariputta (พระสารีบุตร)|พระสารีบุตร]] ณ ฝั่งแม่น้ำคงคา ใกล้**อุกกเจลนคร** แคว้นวัชชี — 16 สูตร (สูตรที่ 2–15 raw ระบุว่า "เหมือนกับในชัมพุขาทกสังยุต" ทุกประการ)

## เนื้อหา

- **สามัณฑกสูตร (ข้อ ๕๑๓)** — สามัณฑกถาม: "นิพพานเป็นไฉน" → พระสารีบุตรตอบ: ความสิ้นราคะ-โทสะ-โมหะ → ถามทางบรรลุ → ตอบ: **อริยมรรคมีองค์ 8** → "เหมือนกับในชัมพุขาทกสังยุต" (สูตร 2–15)
- **ทุกกรปัญหาสูตร (ข้อ ๕๑๔)** — สูตรสุดท้าย: สิ่งทำได้ยาก = บรรพชา; ยากกว่า = ความยินดียิ่ง; ยากกว่า = ปฏิบัติธรรมสมควรแก่ธรรม; ผู้ปฏิบัติจะเป็นอรหันต์ **ไม่นานนัก** (เนื้อหาเหมือนทุกกรปัญหาสูตรในชัมพุขาทกสังยุต)

## ต่างจากชัมพุขาทกสังยุตอย่างไร

| | ชัมพุขาทกสังยุต | สามัณฑกสังยุต |
|---|---|---|
| ผู้ถาม | ชัมพุขาทกปริพาชก | สามัณฑกปริพาชก |
| สถานที่ | บ้านนาลคาม แคว้นมคธ | ฝั่งแม่น้ำคงคา ใกล้อุกกเจลนคร แคว้นวัชชี |
| เนื้อหา | เหมือนกันทุกข้อ | เหมือนกันทุกข้อ |

**ความสำคัญ:** สังยุตนี้สั้นมาก raw บันทึกเฉพาะสูตรที่ 1 และ 16 แล้วระบุว่าสูตร 2–15 "เหมือนชัมพุขาทกสังยุต" — การบันทึกซ้ำเพื่อแสดงว่าพระสารีบุตรตอบคำถามธรรมเดียวกันในคราวและสถานที่ต่างกัน คำตอบไม่เปลี่ยน; โยง [[jambukhadaka-samyutta-page (ชัมพุขาทกสังยุต)|ชัมพุขาทกสังยุต]] สำหรับนิยามธรรม 16 ข้อครบถ้วน; [[sariputta (พระสารีบุตร)|พระสารีบุตร]], [[ariya-magga-8 (อริยมรรคมีองค์ 8)|อริยมรรคมีองค์ 8]]

ทุกขวรรคที่ 6 เป็นวรรคสุดท้ายของ **นิทานสังยุต** (SN 12) ใน **นิทานวรรค** (สังยุตตนิกาย) มี 10 สูตร (ข้อ ๑๘๘–๒๒๙) ตรัสที่[[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]] ทุกสูตร ยกเว้น SN 12.60 ที่กัมมาสทัมมะ วรรคนี้วนซ้ำหัวใจเดิมคือ "เห็นอัสสาทะ (*assāda* ความพอใจ) ในธรรมอันเป็นปัจจัยแห่ง[[samyojana-10 (สังโยชน์ 10)|สังโยชน์]]/[[upadana-4 (อุปาทาน 4)|อุปาทาน]] → [[tanha (ตัณหา)|ตัณหา]]เจริญ → [[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]]สายเกิด; เห็นโทษ → ตัณหาดับ → สายดับ" ผ่านอุปมา 3 ชุด (ไฟกองใหญ่, ประทีปน้ำมัน, ต้นไม้ใหญ่/ต้นไม้อ่อน) และปิดท้ายด้วยสูตรที่แสดงความลึกซึ้งของ[[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]]ต่อ[[ananda (พระอานนท์)|พระอานนท์]]

---

## SN 12.51 ปริวีมังสนสูตร (ข้อ ๑๘๘–๑๙๕) — การพิจารณาเพื่อสิ้นทุกข์โดยชอบ

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

ภิกษุทูลถามว่าต้องพิจารณาด้วยเหตุเท่าใดจึงจะสิ้นทุกข์โดยชอบโดยประการทั้งปวง พระผู้มีพระภาคเจ้าตรัสแสดงการพิจารณาปฏิจจสมุปบาทย้อนจากปลายถึงต้น

- ภิกษุพิจารณา: **ชรามรณะ** มีอะไรเป็นเหตุ? → **ชาติ** → ชาติมีอะไรเป็นเหตุ? → **ภพ** → ภพมีอะไรเป็นเหตุ? → **อุปาทาน** → **ตัณหา** → **เวทนา** → **ผัสสะ** → **สฬายตนะ** → **นามรูป** → **วิญญาณ** → **สังขาร** → มีอะไรเป็นเหตุ? → **[[avijja (อวิชชา)|อวิชชา]]** เป็นต้นตอ
- เมื่อรู้ประจักษ์แต่ละองค์พร้อมเหตุเกิด-ดับและปฏิปทาที่ให้ถึงความดับ และปฏิบัติตามนั้น → ทรงเรียกว่า "ผู้ปฏิบัติเพื่อความสิ้นทุกข์โดยชอบ"
- เมื่อ**ละอวิชชาได้ วิชชาเกิดขึ้น** ภิกษุนั้นไม่ทำสังขาร (บุญ/บาป/อเนญชา) เมื่อไม่ทำ ไม่ถือมั่น ไม่สะดุ้งกลัว ปรินิพพานเฉพาะตน รู้ว่า "ชาติสิ้นแล้ว กิจที่ควรทำทำเสร็จแล้ว"
- อุปมาหม้อร้อน: เมื่อเอาหม้อออกจากเตาวางพื้น ไออุ่นหายไป กระเบื้องหม้อเหลืออยู่ — ภิกษุผู้พ้นแล้วก็รู้ว่าเวทนาทั้งปวงอันตัณหาไม่เพลิดเพลิน "จักเป็นของเย็น สรีรธาตุจักเหลืออยู่ในโลกนี้เท่านั้น"
- โยง: [[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]], [[avijja (อวิชชา)|อวิชชา]], [[vedana (เวทนา)|เวทนา]], [[nibbana (นิพพาน)|นิพพาน]]

## SN 12.52 อุปาทานสูตร (ข้อ ๑๙๖–๑๙๙) — เห็นอัสสาทะ → ตัณหาเจริญ; เห็นโทษ → ตัณหาดับ (อุปมาไฟกองใหญ่)

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

- **สายเกิด**: เมื่อภิกษุเห็นความพอใจ (*assāda*) เนืองๆ ในสังโยชนิยธรรม (*upadāniyesu dhammesu*) → [[tanha (ตัณหา)|ตัณหา]]เจริญ → [[upadana-4 (อุปาทาน 4)|อุปาทาน]] → [[bhava (ภพ)|ภพ]] → ชาติ → ชรามรณะ→กองทุกข์
- **สายดับ**: เมื่อเห็นโทษเนืองๆ → ตัณหาดับ → อุปาทานดับ → ภพดับ → … → กองทุกข์ดับ
- **อุปมาไฟกองใหญ่** (ไม้ 10–40 เล่มเกวียน): บุรุษใส่หญ้าแห้ง/โคมัย/ไม้แห้งเสมอ → ไฟลุกนาน (= เห็นอัสสาทะ); บุรุษไม่ใส่เชื้อ → ไฟดับเพราะสิ้นเชื้อ (= เห็นโทษ)
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[upadana-4 (อุปาทาน 4)|อุปาทาน]]

## SN 12.53 ปฐมสังโยชนสูตร (ข้อ ๒๐๐–๒๐๓) — อุปมาประทีปน้ำมัน (เนื้อหาแล้วอุปมา)

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

หัวข้อเหมือน SN 12.52 แต่เปลี่ยนคำว่า "สังโยชนิยธรรม" (*saṃyojaniyesu dhammesu* ธรรมอันเป็นปัจจัยแห่ง[[samyojana-10 (สังโยชน์ 10)|สังโยชน์]]) แทน "อุปาทานิยธรรม" และเปลี่ยนอุปมา

- **อุปมาประทีปน้ำมัน**: บุรุษเติมน้ำมัน/ใส่ไส้เสมอ → ประทีปลุกตลอดกาล (= เห็นอัสสาทะ → ตัณหาเจริญ); บุรุษไม่เติมน้ำมัน/ไม่ใส่ไส้ → ประทีปดับเพราะสิ้นเชื้อ (= เห็นโทษ → ตัณหาดับ)
- สูตรนี้แสดงเนื้อหาก่อน จึงอุปมาทีหลัง (ต่างจาก SN 12.54)
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[samyojana-10 (สังโยชน์ 10)|สังโยชน์]]

---

## SN 12.54 ทุติยสังโยชนสูตร (ข้อ ๒๐๔–๒๐๕) — อุปมาประทีปน้ำมัน (อุปมาก่อนแล้วเนื้อหา)

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

เนื้อหาและอุปมาเหมือน SN 12.53 ทุกประการ ต่างเพียงลำดับการแสดง — SN 12.54 ตรัส**อุปมาก่อน แล้วจึงตรัสเนื้อหา** (ต่างจาก SN 12.53 ที่ตรัสเนื้อหาก่อน)

- คู่สูตร SN 12.53–12.54 แสดงให้เห็นว่าธรรมบทเดียวกันนำเสนอในลำดับต่างกันได้ทั้งสองทาง
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[samyojana-10 (สังโยชน์ 10)|สังโยชน์]]

---

## SN 12.55 ปฐมมหารุกขสูตร (ข้อ ๒๐๖–๒๐๙) — อุปมาต้นไม้ใหญ่ (เนื้อหาก่อน)

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

เปลี่ยนกลับมาใช้ "อุปาทานิยธรรม" เหมือน SN 12.52 และเปลี่ยนอุปมาเป็น **ต้นไม้ใหญ่**

- **อุปมาต้นไม้ใหญ่ (สายเกิด)**: ต้นไม้ใหญ่มีรากหยั่งลงและแผ่ข้างๆ รากดูดโอชารสไปเบื้องบน → ต้นไม้เป็นอยู่ตลอดกาลนาน = เมื่อเห็นอัสสาทะในอุปาทานิยธรรม → [[tanha (ตัณหา)|ตัณหา]]เจริญ → [[upadana-4 (อุปาทาน 4)|อุปาทาน]] → ภพ → … → กองทุกข์
- **อุปมาต้นไม้ใหญ่ (สายดับ)**: บุรุษเอาจอบขุดต้นไม้ที่โคน คุ้ยรากใหญ่เล็กแม้เท่าก้านแฝก ทอนเป็นท่อนผ่าเกรียกเป็นชิ้นผึ่งลมตากแดดเผาไฟเป็นเขม่าโปรยลมหรือลอยแม่น้ำ → ต้นไม้ถูกตัดรากขึ้น ถึงความไม่มี ไม่เกิดอีก = เมื่อเห็นโทษ → ตัณหาดับ → กองทุกข์ดับ
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[upadana-4 (อุปาทาน 4)|อุปาทาน]]

---

## SN 12.56 ทุติยมหารุกขสูตร (ข้อ ๒๑๐–๒๑๑) — อุปมาต้นไม้ใหญ่ (อุปมาก่อน)

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

เนื้อหาและอุปมาเหมือน SN 12.55 ทุกประการ ต่างเพียงลำดับ — SN 12.56 ตรัส**อุปมาต้นไม้ใหญ่ก่อน แล้วจึงตรัสเนื้อหา**

- คู่สูตร SN 12.55–12.56 คล้องกับคู่ SN 12.53–12.54 คือธรรมเดียวกัน นำเสนอในลำดับสลับกัน
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[upadana-4 (อุปาทาน 4)|อุปาทาน]]

---

## SN 12.57 ตรุณรุกขสูตร (ข้อ ๒๑๒–๒๑๕) — อุปมาต้นไม้อ่อน

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

เปลี่ยนอุปมาเป็น **ต้นไม้อ่อน** (*taruṇa*) และใช้ "สังโยชนิยธรรม" เหมือน SN 12.53–12.54

- **อุปมาต้นไม้อ่อน (สายเกิด)**: บุรุษพรวนดินใส่ปุ๋ยรดน้ำเสมอ → ต้นไม้อ่อนเจริญงอกงามไพบูลย์ = เห็นอัสสาทะในสังโยชนิยธรรม → [[tanha (ตัณหา)|ตัณหา]]เจริญ → [[upadana-4 (อุปาทาน 4)|อุปาทาน]] → ภพ → … → กองทุกข์
- **อุปมาต้นไม้อ่อน (สายดับ)**: บุรุษตัดรากและทำลายเหมือน SN 12.55 → ต้นไม้อ่อนถึงความไม่มี = เห็นโทษ → ตัณหาดับ → กองทุกข์ดับ
- สูตรนี้เสนอเนื้อหาก่อน แล้วจึงอุปมา
- โยง: [[tanha (ตัณหา)|ตัณหา]], [[samyojana-10 (สังโยชน์ 10)|สังโยชน์]]

---

## SN 12.58 นามรูปสูตร (ข้อ ๒๑๖–๒๑๙) — เห็นอัสสาทะ → นามรูปหยั่งลง

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

สูตรนี้แตกต่างจาก SN 12.52–12.57 ตรงที่ไม่ใช้ตัณหาเป็นจุดเชื่อม แต่แสดงว่าการเห็นอัสสาทะส่งผลไปที่ **[[namarupa (นามรูป)|นามรูป]]** โดยตรง

- **สายเกิด**: เมื่อเห็นความพอใจเนืองๆ ในสังโยชนิยธรรม → **นามรูปหยั่งลง** (*okkamati*) → เพราะนามรูปเป็นปัจจัยจึงมี[[salayatana (สฬายตนะ)|สฬายตนะ]] → [[phassa (ผัสสะ)|ผัสสะ]] → [[vedana (เวทนา)|เวทนา]] → [[tanha (ตัณหา)|ตัณหา]] → [[upadana-4 (อุปาทาน 4)|อุปาทาน]] → ภพ → ชาติ → ชรามรณะ → กองทุกข์
- **สายดับ**: เมื่อเห็นโทษเนืองๆ → **นามรูปไม่หยั่งลง** → นามรูปดับ → สฬายตนะดับ → … → กองทุกข์ดับ
- ใช้อุปมาต้นไม้ใหญ่เหมือน SN 12.55 (เนื้อหาก่อน อุปมาทีหลัง)
- โยง: [[namarupa (นามรูป)|นามรูป]], [[salayatana (สฬายตนะ)|สฬายตนะ]], [[tanha (ตัณหา)|ตัณหา]]

---

## SN 12.59 วิญญาณสูตร (ข้อ ๒๒๐–๒๒๓) — เห็นอัสสาทะ → วิญญาณหยั่งลง

สถานที่: [[jetavana (เชตวัน)|พระเชตวัน]] [[savatthi (สาวัตถี)|สาวัตถี]]

คู่กับ SN 12.58 เปลี่ยนองค์ที่ "หยั่งลง/ไม่หยั่งลง" เป็น **[[vinnana (วิญญาณ)|วิญญาณ]]**

- **สายเกิด**: เมื่อเห็นความพอใจเนืองๆ ในสังโยชนิยธรรม → **วิญญาณหยั่งลง** → เพราะวิญญาณเป็นปัจจัยจึงมี[[namarupa (นามรูป)|นามรูป]] → สฬายตนะ → ผัสสะ → เวทนา → ตัณหา → อุปาทาน → ภพ → ชาติ → ชรามรณะ → กองทุกข์
- **สายดับ**: เมื่อเห็นโทษเนืองๆ → **วิญญาณไม่หยั่งลง** → วิญญาณดับ → นามรูปดับ → … → กองทุกข์ดับ
- ใช้อุปมาต้นไม้ใหญ่เหมือน SN 12.55–12.58
- คู่สูตร SN 12.58–12.59 แสดงว่าจุดที่ "หยั่งลง" (okkamati) ได้แก่ทั้งนามรูปและวิญญาณ สะท้อนความสัมพันธ์วนกันระหว่าง[[vinnana (วิญญาณ)|วิญญาณ]]และ[[namarupa (นามรูป)|นามรูป]]ในปฏิจจสมุปบาท
- โยง: [[vinnana (วิญญาณ)|วิญญาณ]], [[namarupa (นามรูป)|นามรูป]], [[tanha (ตัณหา)|ตัณหา]]

## SN 12.60 นิทานสูตร (ข้อ ๒๒๔–๒๒๙) — ปฏิจจสมุปบาทลึกซึ้ง อย่าประมาท

สถานที่: **กัมมาสทัมมะ** (*Kammāsadamma*) นิคมของชาวกุรุ [[kammasadhamma (กัมมาสทัมมะ)|แคว้นกุรุรัฐ]]

สูตรนี้แตกต่างจาก 9 สูตรก่อนทั้งที่ตั้งและผู้สนทนา — [[ananda (พระอานนท์)|ท่านพระอานนท์]]เข้าเฝ้าแล้วกราบทูลว่า:

**"น่าอัศจรรย์ พระเจ้าข้า ปฏิจจสมุปบาทนี้เป็นธรรมลึกซึ้งเพียงไร ทั้งมีกระแสความลึกซึ้ง แต่ถึงอย่างนั้น ก็ปรากฏเหมือนธรรมง่ายๆ แก่ข้าพระองค์"**

- พระผู้มีพระภาคเจ้าตรัสห้ามว่า **"อย่ากล่าวอย่างนี้ อานนท์ อย่ากล่าวอย่างนี้"** — ปฏิจจสมุปบาทนี้เป็นธรรมลึกซึ้ง ทั้งมีกระแสความลึกซึ้ง เพราะ**ไม่รู้ ไม่ตรัสรู้ ไม่แทงตลอดธรรมนี้** หมู่สัตว์จึงเป็นเหมือนเส้นด้ายที่ยุ่ง เหมือนกลุ่มเส้นด้ายเป็นปม เหมือนหญ้ามุงกระต่ายและหญ้าปล้อง ย่อมไม่ผ่านพ้นอบาย ทุคติ วินิบาต สงสาร
- จากนั้นทรงแสดง[[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]]สายเกิด-ดับแก่[[ananda (พระอานนท์)|พระอานนท์]] ด้วยอุปมาต้นไม้ใหญ่ (เห็นอัสสาทะในอุปาทานิยธรรม → ตัณหาเจริญ/ดับ → กองทุกข์เกิด/ดับ)
- ความสำคัญ: สูตรนี้เป็นคำเตือนที่สำคัญ — แม้แต่[[ananda (พระอานนท์)|พระอานนท์]]ผู้ทรงจำพระธรรมมากที่สุดก็ยังประเมินความลึกของปฏิจจสมุปบาทต่ำเกินไป พระผู้มีพระภาคเจ้าทรงย้ำว่าความลึกซึ้งนี้เป็นเหตุให้สัตว์วนสงสาร ผู้ไม่แทงตลอดย่อมผ่านพ้นอบายและสงสารไม่ได้
- โยง: [[ananda (พระอานนท์)|พระอานนท์]], [[paticcasamuppada (ปฏิจจสมุปบาท)|ปฏิจจสมุปบาท]], [[kammasadhamma (กัมมาสทัมมะ)|กัมมาสทัมมะ]], [[tanha (ตัณหา)|ตัณหา]], [[upadana-4 (อุปาทาน 4)|อุปาทาน]]

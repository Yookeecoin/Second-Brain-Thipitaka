# Second Brain — พระไตรปิฎก Wiki (Obsidian-Dhamma)

knowledge base ส่วนตัวเรื่อง**พระไตรปิฎก** (Tipiṭaka) ในรูปแบบ **"LLM Wiki"** — ผู้ใช้เป็นคนหาแหล่ง อ่าน ตั้งคำถาม ส่วน LLM (Claude) เป็นผู้สรุป เชื่อมโยง (cross-reference) จัดหมวด และดูแล wiki ทั้งหมด

ต่างจาก RAG ที่ค้นใหม่ทุกครั้ง — ระบบนี้ **สะสมความรู้ทบต้น**: ทุกครั้งที่เพิ่มแหล่งใหม่ ความรู้จะถูก integrate เข้ากับหน้าเดิม ปรับสรุป โยงข้าม และ flag จุดที่ขัดกัน

> 📍 **ความคืบหน้าปัจจุบัน:** มัชฌิมนิกาย 64/152 สูตร (กำลัง ingest พระสุตตันตปิฎกต่อเนื่อง)

## โครงสร้าง

```
second-brain/
├── CLAUDE.md          # schema + กติกาการทำงานของ LLM (หัวใจของระบบ)
├── wiki/              # ★ หน้าที่ LLM สร้างทั้งหมด (ผลงานหลัก)
│   ├── index.md       #   สารบัญทุกหน้า จัดตามหมวด
│   ├── log.md         #   บันทึกเหตุการณ์ตามเวลา
│   ├── suttas/        #   หน้าพระสูตรแต่ละสูตร
│   ├── persons/       #   บุคคล (พระพุทธเจ้า สาวก บุคคลในเรื่อง)
│   ├── places/        #   สถานที่ (เมือง วัด แคว้น)
│   ├── dhamma/        #   หลักธรรม/concept (อริยสัจ 4, ปฏิจจสมุปบาท ฯลฯ)
│   └── terms/         #   อภิธานศัพท์บาลี / อ้างอิง
├── raw/               # ต้นฉบับพระไตรปิฎก (immutable, อ่านอย่างเดียว)
└── export/            # ไฟล์ที่ export ไปใช้กับเครื่องมืออื่น
```

## วิธีใช้

1. เปิดโฟลเดอร์นี้เป็น **Obsidian vault** — หน้าใน `wiki/` โยงกันด้วย `[[wikilink]]` ดูเป็นกราฟความรู้ได้
2. แต่ละหน้าขึ้นต้นด้วย YAML frontmatter (title/type/tags/ref...) ตามด้วยเนื้อหาภาษาไทย เก็บศัพท์บาลีในวงเล็บ
3. การทำงานทั้งหมดเดินตาม schema ใน [`CLAUDE.md`](CLAUDE.md) (INGEST / QUERY / LINT)

## หลักความถูกต้อง

- **ไม่แต่งพุทธพจน์ ไม่กุเลขที่อ้างอิง** — ถ้าไม่แน่ใจระบุว่า "ยังไม่ระบุที่มา"
- แยกชัดระหว่าง **เนื้อความในพระสูตร** กับ **คำอธิบาย/ตีความของอรรถกถา**ภายหลัง

## 🙏 ที่มาต้นฉบับ & ลิขสิทธิ์

- ต้นฉบับพระไตรปิฎกใน `raw/tipitaka/thaimm/` อ้างอิงจากพระไตรปิฎกและอรรถกถาแปล ฉบับ **มหามกุฏราชวิทยาลัย (มมร)** — **ลิขสิทธิ์เป็นของมูลนิธิมหามกุฏราชวิทยาลัย** นำมาใช้เพื่อการศึกษาส่วนบุคคล กรุณาเคารพลิขสิทธิ์ของเจ้าของต้นฉบับ
- ส่วนเนื้อหาใน `wiki/` (บทสรุป การเชื่อมโยง การจัดหมวด) เป็นงานเรียบเรียงที่สร้างโดย LLM ภายใต้การกำกับของผู้ดูแล vault

> หมายเหตุ: หากเจ้าของลิขสิทธิ์ต้นฉบับประสงค์ให้นำส่วน `raw/` ออก ยินดีดำเนินการทันที

*จัดทำเป็น knowledge base ส่วนตัวเพื่อการศึกษาพระธรรม — เผยแผ่เป็นธรรมทาน*
